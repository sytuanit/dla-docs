# 임시 수입

## 1. 목적

**임시 수입** 모듈은 다음과 같은 고정 주기가 없는 비정기적인 수입을 기록하는 데 도움이 됩니다:
- 온라인 판매
- 프리랜스
- 보너스
- 현금 선물
- 기타 불규칙한 수입

**정기 수입**과 달리, 임시 수입에는 자동 주기가 없으며 매번 수동으로 입력해야 합니다.

## 2. 사용 시기

다음과 같은 경우 이 모듈을 사용합니다:
- 무작위이고 비정기적인 수입 기록
- 기간 내 총 수입 추적
- 임시 수입 추세 분석
- 월별 예산에 계산

## 3. 관련 화면

- 임시 수입 목록
- 새 수입 추가
- 수입 편집

## 4. 주요 사용 방법

### 4.1 임시 수입 추가

1. **기능** → **임시 수입** 선택
2. 오른쪽 하단의 **+** (FAB) 버튼 탭
3. 정보 입력:
   - **카테고리**: 카테고리 선택 또는 새로 만들기
   - **금액**: 수령액 입력
   - **날짜**: 돈을 받은 날 선택 (기본값은 오늘)
   - **메모**: 상세 설명 (선택 사항)
4. **저장** 탭

### 4.2 수입 목록 표시

1. **기능** → **임시 수입** 선택
2. 목록이 설정된 레이아웃(1, 2, 3 또는 4열)에 따라 표시됩니다
3. **검색**을 사용하여 카테고리 또는 메모로 필터링
4. **시간 필터** 선택: 오늘 / 이번 주 / 이번 달 / 지난 달 / 사용자 지정

### 4.3 수입 편집

1. 임시 수입 목록으로 이동
2. 편집할 항목 길게 누르기
3. 메뉴에서 **편집** 선택
4. 정보 업데이트
5. **저장** 탭

### 4.4 수입 삭제

1. 임시 수입 목록으로 이동
2. 삭제할 항목 길게 누르기
3. 메뉴에서 **삭제** 선택
4. 삭제 확인

## 6. 로직 및 규칙

### 6.1 표시 레이아웃

- 열 수를 설정할 수 있습니다: 1, 2, 3 또는 4열
- 레이아웃은 설정에 저장되며 모든 임시 수입 목록에 적용됩니다

### 6.2 시간 필터

- **오늘**: 오늘의 수입만 표시
- **이번 주**: 주 시작부터 오늘까지
- **이번 달**: 달 시작부터 오늘까지
- **지난 달**: 전월 전체
- **사용자 지정**: 사용자 지정 시간 범위 선택

### 6.3 검색

- **카테고리명**과 **메모**로 검색
- 대소문자 구분 없음
- 입력에 따라 실시간 검색

### 6.4 예산 통합

- 임시 수입은 예산의 "임시 수입"에 계산됩니다
- 월별 총 수입 추적에 도움이 됩니다

## 7. 중요 참고 사항

- **주기 없음**: 임시 수입에는 자동 주기가 없으며 매번 수동으로 입력해야 합니다
- **삭제 가능**: 모든 수입을 삭제할 수 있습니다
- **예산 통합**: 임시 수입은 현재 달의 예산에 자동 계산됩니다
