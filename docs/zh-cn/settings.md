# 设置

## 1. 目的

**设置**模块允许您根据个人需求配置应用程序，包括：
- 语言和货币
- 通知
- 类别
- 备份和还原数据
- 安全性（密码、Face ID）

## 2. 使用时机

当您想：
- 变更语言或货币时
- 启用/停用通知时
- 管理类别（新增、编辑、删除、设为默认）时
- 备份或还原数据时
- 变更密码或启用 Face ID 时

## 3. 相关画面

- 主要设置画面
- 基本设置
- 选择语言
- 选择货币
- 管理类别
- 备份数据
- 还原数据
- 变更密码
- 启用 Face ID / 指纹

## 4. 主要使用方式

### 4.1 变更语言

1. 前往**设置** → **显示与语言** → **语言**
2. 选择所需语言
3. 应用程序将自动以新语言重新加载

### 4.2 变更货币

1. 前往**设置** → **显示与语言** → **货币**
2. 选择货币类型
3. 所有金额将以新单位显示

### 4.3 启用/停用通知

1. 前往**设置** → **通知**
2. 切换**通知**开关
3. 如果启用，您将收到关于以下事项的提醒：
   - 定期收入到期
   - 定期支出到期
   - 储蓄账户到期
   - 特殊场合即将到来

### 4.4 管理类别

1. 前往**设置** → **类别**
2. 选择要管理的类别类型：
   - 定期收入
   - 额外收入
   - 定期支出
   - 日常支出
3. 新增/编辑/删除类别
4. 设定默认类别（用于日常支出）

### 4.5 备份数据

1. 前往**设置** → **备份与数据** → **备份**
2. 选择保存位置（文件系统）
3. 点击**备份**
4. 将创建备份文件

### 4.6 还原数据

1. 前往**设置** → **备份与数据** → **还原**
2. 选择备份文件
3. 确认还原
4. **注意**：还原将覆盖当前数据

## 6. 逻辑与规则

### 6.1 语言

- 支持：English、Tiếng Việt、日本語、简体中文
- 变更语言将重新加载整个应用程序
- 系统类别将自动翻译为新语言

### 6.2 货币

- 每种语言都有默认货币（例如：CNY 用于简体中文）
- 您可以选择不同货币
- 所有金额将根据选定的货币格式化

### 6.3 通知

- 通知仅在应用程序有权限时运作
- 通知时间取决于每个功能，可设定：
  - 定期收入/支出：`notificationTime1`、`notificationTime2`（默认 16:00 和 19:00）
  - 储蓄账户和贷款：`notificationTime1`、`notificationTime2`（默认 10:00 和 19:00）
  - 特殊场合和准备步骤：根据您输入的 `reminderTime`
  - 可以分别停用每种类型的通知（未来功能）

### 6.4 类别

- 系统类别无法删除，只能停用
- 用户类别可以删除（如未使用）
- 每种类别类型都是独立的（定期收入、日常支出等）

## 7. 重要注意事项

- **定期备份**：应定期备份数据以避免数据丢失
- **还原将覆盖**：还原将取代所有当前数据
- **密码**：如果您忘记密码，可以重置（将删除数据）
