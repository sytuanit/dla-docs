# ລາຍງານ

## 1. ຈຸດປະສົງ

ໂມດູນ **ລາຍງານ** ຊ່ວຍທ່ານເບິ່ງສະຫຼຸບການເງິນລາຍລະອຽດ, ລວມທັງ:
- ສະຫຼຸບສຸຂະພາບການເງິນ
- ການວິເຄາະລາຍໄດ້/ລາຍຈ່າຍ
- ຄວາມຄືບໜ້າການອອມເງິນ
- ລາຍງານເງິນກູ້
- ການຄາດຄະເນການເງິນ

## 2. ເມື່ອໃດທີ່ຄວນໃຊ້

ໃຊ້ໂມດູນນີ້ເມື່ອທ່ານຕ້ອງການ:
- ເບິ່ງສະຫຼຸບການເງິນລວມ
- ວິເຄາະລາຍໄດ້/ລາຍຈ່າຍຕາມປະເພດ
- ຕິດຕາມຄວາມຄືບໜ້າການອອມເງິນ
- ເບິ່ງລາຍງານເງິນກູ້
- ຄາດຄະເນການເງິນໃນອະນາຄົດ

## 3. ຫນ້າຈໍທີ່ກ່ຽວຂ້ອງ

- ຫນ້າຈໍລາຍງານຫຼັກ
- ລາຍງານລາຍໄດ້
- ລາຍງານລາຍຈ່າຍ
- ລາຍງານງົບປະມານ
- ລາຍງານການອອມເງິນ
- ລາຍງານເງິນກູ້

## 4. ການນໍາໃຊ້ຫຼັກ

### 4.1 ເບິ່ງສະຫຼຸບສຸຂະພາບການເງິນ

1. ໄປທີ່ **ຟັງຊັນ** → ເລືອກ **ລາຍງານ**
2. ເບິ່ງຫນ້າຈໍລາຍງານຫຼັກ
3. ເບິ່ງສະຫຼຸບ:
   - ລາຍໄດ້ລວມ
   - ລາຍຈ່າຍລວມ
   - ການອອມເງິນ
   - ສຸຂະພາບການເງິນ

### 4.2 ເບິ່ງລາຍງານລາຍໄດ້

1. ໄປທີ່ຫນ້າຈໍລາຍງານຫຼັກ
2. ເລືອກ **ລາຍງານລາຍໄດ້**
3. ເບິ່ງການວິເຄາະລາຍໄດ້ຕາມປະເພດ
4. ເບິ່ງລາຍການລາຍໄດ້ລາຍລະອຽດ

### 4.3 ເບິ່ງລາຍງານລາຍຈ່າຍ

1. ໄປທີ່ຫນ້າຈໍລາຍງານຫຼັກ
2. ເລືອກ **ລາຍງານລາຍຈ່າຍ**
3. ເບິ່ງການວິເຄາະລາຍຈ່າຍຕາມປະເພດ
4. ເບິ່ງລາຍການລາຍຈ່າຍລາຍລະອຽດ

### 4.4 ເບິ່ງລາຍງານງົບປະມານ

1. ໄປທີ່ຫນ້າຈໍລາຍງານຫຼັກ
2. ເລືອກ **ລາຍງານງົບປະມານ**
3. ເບິ່ງສະຫຼຸບງົບປະມານເດືອນປັດຈຸບັນ
4. ເບິ່ງການວິເຄາະການໃຊ້ຈ່າຍຕາມປະເພດ

### 4.5 ເບິ່ງລາຍງານການອອມເງິນ

1. ໄປທີ່ຫນ້າຈໍລາຍງານຫຼັກ
2. ເລືອກ **ລາຍງານການອອມເງິນ**
3. ເບິ່ງຄວາມຄືບໜ້າການອອມເງິນ
4. ເບິ່ງລາຍການບັນຊີອອມເງິນ

### 4.6 ເບິ່ງລາຍງານເງິນກູ້

1. ໄປທີ່ຫນ້າຈໍລາຍງານຫຼັກ
2. ເລືອກ **ລາຍງານເງິນກູ້**
3. ເບິ່ງສະຫຼຸບເງິນກູ້
4. ເບິ່ງຕາຕະລາງການຊໍາລະ

## 5. ຕົວຢ່າງ & ການສະແດງຜົນ UI

### REPORT-01: ເບິ່ງສະຫຼຸບສຸຂະພາບການເງິນ

**ເປົ້າຫມາຍ**: ເບິ່ງສະຫຼຸບການເງິນລວມເພື່ອປະເມີນສຸຂະພາບການເງິນຂອງທ່ານ.

**ຂັ້ນຕອນ**:
1. ໄປທີ່ຟັງຊັນ → ລາຍງານ
2. ເບິ່ງສະຫຼຸບສຸຂະພາບການເງິນ
3. ເບິ່ງລາຍໄດ້ລວມ, ລາຍຈ່າຍລວມ, ການອອມເງິນ

---

### REPORT-02: ເບິ່ງລາຍງານລາຍໄດ້ລາຍລະອຽດ

**ເປົ້າຫມາຍ**: ເບິ່ງການວິເຄາະລາຍໄດ້ຕາມປະເພດແລະເວລາ.

**ຂັ້ນຕອນ**:
1. ໄປທີ່ຫນ້າຈໍລາຍງານຫຼັກ
2. ເລືອກ "ລາຍງານລາຍໄດ້"
3. ເບິ່ງການວິເຄາະລາຍໄດ້ຕາມປະເພດ
4. ເບິ່ງລາຍການລາຍໄດ້ລາຍລະອຽດ

## 6. ເຫດຜົນ & ກົດລະບຽບ

### 6.1 ການຄິດໄລ່

- **ລາຍໄດ້ລວມ**: ລາຍໄດ້ປົກກະຕິ + ລາຍໄດ້ເພີ່ມເຕີມ
- **ລາຍຈ່າຍລວມ**: ລາຍຈ່າຍປົກກະຕິ + ລາຍຈ່າຍປະຈໍາວັນ
- **ການອອມເງິນ**: ລາຍໄດ້ - ລາຍຈ່າຍ

### 6.2 ການວິເຄາະ

- ການວິເຄາະຕາມປະເພດ: ສະແດງລາຍໄດ້/ລາຍຈ່າຍຕາມປະເພດ
- ການວິເຄາະຕາມເວລາ: ສະແດງລາຍໄດ້/ລາຍຈ່າຍຕາມເດືອນ

### 6.3 ການຄາດຄະເນ

- ການຄາດຄະເນຖືກຄິດໄລ່ຕາມລາຍໄດ້ແລະລາຍຈ່າຍປັດຈຸບັນ
- ສະແດງການຄາດຄະເນສໍາລັບ 12, 24, 36 ເດືອນ

## 7. ໝາຍເຫດສໍາຄັນ

- **ຂໍ້ມູນບໍ່ພຽງພໍ**: ຖ້າບໍ່ມີຂໍ້ມູນພຽງພໍ, ລາຍງານຈະສະແດງຂໍ້ຄວາມ "ບໍ່ມີຂໍ້ມູນພຽງພໍ"
- **ການອັບເດດອັດຕະໂນມັດ**: ລາຍງານຖືກອັບເດດອັດຕະໂນມັດເມື່ອມີຂໍ້ມູນໃໝ່
- **ການກອງ**: ສາມາດກອງລາຍງານຕາມເວລາ (ມື້ນີ້, ອາທິດນີ້, ເດືອນນີ້, ແລະອື່ນໆ)
