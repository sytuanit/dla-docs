# 财务目标

## 1. 目的

**财务目标**模块可帮助您：
- 设定财务目标（例如：买房、买车）
- 规划财务以实现目标
- 评估财务假设
- 比较贷款方案
- 追踪目标进度

## 2. 使用时机

当您想：
- 规划重大财务目标时
- 评估购买资产的借贷能力时
- 比较财务选项时
- 追踪储蓄进度时

## 3. 相关画面

- 创建财务目标（3 个步骤）
- 目标详细信息和计划
- 查看贷款计划
- 评估假设
- 评估假设和贷款

## 4. 主要使用方式

### 4.1 创建财务目标（3 个步骤）

#### 步骤 1：输入财务计划

1. 前往**功能** → 选择**规划与假设**
2. 点击 **➕ 新增** (FAB) 按钮
3. 查看自动填入的信息：
   - **平均收入**：从启用中的定期收入自动检索（可点击查看明细）
   - **固定支出**：从启用中的定期支出 + 贷款付款自动检索（可点击查看明细）
   - **当前余额**：从当前余额自动检索
4. 输入**生活支出**：每月生活支出（饮食、交通等）
5. 查看自动计算的预测：
   - 12 个月后
   - 24 个月后
   - 36 个月后
   （如果保持当前收入和支出水平）
6. 点击**继续**

#### 步骤 2：输入目标信息

1. 输入**目标名称**：（例如："买房"）
2. 输入**所需金额**：实现目标所需的总金额
3. 查看**首付**：从当前余额自动填入（可编辑）
4. 点击**继续**

#### 步骤 3：检查目标实现能力

1. 查看目标信息：名称、目标价值、首付、剩余缺口
2. 查看当前财务状况：平均收入、平均支出、平均储蓄
3. 查看结论：
   - "您将在约 X 年内实现目标"（如果储蓄 > 0）
   - "在当前情况下，不借贷或改善财务状况，您无法实现目标"（如果储蓄 <= 0）
4. 查看下一步选项：
   - **查看贷款选项**：评估借贷能力
   - **创建收入/支出假设**：模拟财务改善
   - **结合假设 + 贷款**：最佳方案
5. 点击**保存目标**（可以现在保存或稍后创建计划）

### 4.2 查看目标清单和详细信息

1. 前往**功能** → 选择**规划与假设**
2. 查看已创建的目标清单：
   - 每个目标显示：名称、目标价值、已付首付、剩余缺口
3. （可选）使用搜索栏依名称查找目标
4. 点击目标查看详细信息：
   - **目标信息**：名称、目标价值、首付、剩余缺口
   - **财务计划（基线）**：点击查看带有平均收入、支出、储蓄的对话框
   - **已保存计划清单**：已创建的贷款计划、假设或组合

### 4.3 为目标创建贷款计划

1. 在目标详细信息画面上，点击**➕ 新增**按钮
2. 应用程序显示选择计划类型的对话框，选择**"贷款"**
3. 输入贷款信息：贷款金额、利率、贷款期限、计划名称
4. 查看自动计算结果：每月付款、总付款金额、实现目标的时间、可负担性
5. 点击**保存计划**

### 4.4 创建收入/支出假设

1. 在目标详细信息画面上，点击**➕ 新增**按钮
2. 应用程序显示选择计划类型的对话框，选择**"假设"**
3. 输入假设：
   - **增加收入**：额外金额（如无增加则留空）
   - **减少支出**：减少金额（如无减少则留空）
   - 假设名称
4. 查看自动计算结果：新收入、新支出、新储蓄、实现目标的新时间
5. 点击**保存假设**

### 4.5 删除财务目标

1. 前往目标详细信息画面
2. 点击目标卡片右上角的**删除**按钮（删除图标）
3. 应用程序显示确认对话框
4. 点击**删除**以确认

**注意**：删除目标将删除所有相关计划且无法撤销。

## 6. 逻辑与规则

### 6.1 预测计算

- 预测基于：
  - 收入 - 固定支出 - 生活支出 = 储蓄/月
  - 当前余额 + (储蓄/月 × 月数)

### 6.2 目标

- 剩余缺口 = 所需金额 - 首付
- 估计时间 = 剩余缺口 / 平均储蓄（月）
- 如果平均储蓄 <= 0：不借贷或改善财务状况，无法实现目标

### 6.3 贷款计划

- 计算基于：
  - 贷款金额
  - 利率
  - 期限
  - 自动创建付款时间表

### 6.4 财务假设

- 评估假设，例如：
  - 增加/减少收入
  - 增加/减少支出
  - 更改利率
- 查看对目标实现能力的影响

## 7. 重要注意事项

- **需要高级模块**：此功能仅供高级用户使用
- **预测仅供参考**：基于收入和支出稳定的假设
- **可创建多个计划**：您可以创建多个计划（贷款、假设、组合）进行比较
- **基线已保存**：创建目标时保存初始财务计划（基线），用于与后续计划比较
- **自动计算**：收入和固定支出从启用中的定期项目自动检索，包括银行贷款付款
- **删除目标**：删除目标时，所有相关计划也会被删除且无法恢复
