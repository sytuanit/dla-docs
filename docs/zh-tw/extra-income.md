# 額外收入

## 1. 目的

**額外收入**模組可協助您記錄非定期、無固定週期的收入，例如：
- 線上銷售
- 自由工作
- 獎金
- 現金禮物
- 其他不規則收入

與**定期收入**不同，額外收入沒有自動週期，您必須每次手動輸入。

## 2. 使用時機

當您想：
- 記錄隨機、非定期收入時
- 追蹤某段時間的總收入時
- 分析額外收入趨勢時
- 計算到每月預算時

## 3. 相關畫面

- 額外收入清單
- 新增收入
- 編輯收入

## 4. 主要使用方式

### 4.1 新增額外收入

1. 前往**功能** → 選擇**額外收入**
2. 點擊右下角的 **+** (FAB) 按鈕
3. 填寫資訊：
   - **類別**：選擇或建立新類別
   - **金額**：輸入收到的金額
   - **日期**：選擇收到錢的日期（預設為今天）
   - **備註**：詳細說明（選填）
4. 點擊**儲存**

### 4.2 查看收入清單

1. 前往**功能** → 選擇**額外收入**
2. 清單根據您設定的版面配置顯示（1、2、3 或 4 欄）
3. 使用**搜尋**依類別或備註篩選
4. 選擇**時間篩選**：今天 / 本週 / 本月 / 上月 / 自訂

### 4.3 編輯收入

1. 前往額外收入清單
2. 長按要編輯的項目
3. 從選單中選擇**編輯**
4. 更新資訊
5. 點擊**儲存**

### 4.4 刪除收入

1. 前往額外收入清單
2. 長按要刪除的項目
3. 從選單中選擇**刪除**
4. 確認刪除

## 6. 邏輯與規則

### 6.1 顯示版面

- 您可以設定欄數：1、2、3 或 4 欄
- 版面配置儲存在設定中，適用於所有額外收入清單

### 6.2 時間篩選

- **今天**：僅顯示今天的收入
- **本週**：從週開始到今天
- **本月**：從月初到今天
- **上月**：整個上個月
- **自訂**：選擇自訂時間範圍

### 6.3 搜尋

- 在**類別名稱**和**備註**中搜尋
- 不區分大小寫
- 輸入時即時搜尋

### 6.4 預算整合

- 額外收入會計算到預算中的「額外收入」
- 協助您追蹤每月總收入

## 7. 重要注意事項

- **無週期**：額外收入沒有自動週期，您必須每次手動輸入
- **可刪除**：您可以刪除任何收入
- **預算整合**：額外收入會自動計算到當月預算中
