# 設定

## 1. 目的

**設定**模組允許您根據個人需求配置應用程式，包括：
- 語言和貨幣
- 通知
- 類別
- 備份和還原資料
- 安全性（密碼、Face ID）

## 2. 使用時機

當您想：
- 變更語言或貨幣時
- 啟用/停用通知時
- 管理類別（新增、編輯、刪除、設為預設）時
- 備份或還原資料時
- 變更密碼或啟用 Face ID 時

- 還原資料
- 變更密碼
- 啟用 Face ID / 指紋

## 4. 主要使用方式

### 4.1 變更語言

1. 前往**設定** → **顯示與語言** → **語言**
2. 選擇所需語言
3. 應用程式將自動以新語言重新載入

### 4.2 變更貨幣

1. 前往**設定** → **顯示與語言** → **貨幣**
2. 選擇貨幣類型
3. 所有金額將以新單位顯示

### 4.3 啟用/停用通知

1. 前往**設定** → **通知**
2. 切換**通知**開關
3. 如果啟用，您將收到關於以下事項的提醒：
   - 定期收入到期
   - 定期支出到期
   - 儲蓄帳戶到期
   - 特殊場合即將到來

### 4.4 管理類別

1. 前往**設定** → **類別**
2. 選擇要管理的類別類型：
   - 定期收入
   - 額外收入
   - 定期支出
   - 日常支出
3. 新增/編輯/刪除類別
4. 設定預設類別（用於日常支出）

### 4.5 備份資料

1. 前往**設定** → **備份與資料** → **備份**
2. 選擇儲存位置（檔案系統）
3. 點擊**備份**
4. 將建立備份檔案

### 4.6 還原資料

1. 前往**設定** → **備份與資料** → **還原**
2. 選擇備份檔案
3. 確認還原
4. **注意**：還原將覆蓋目前資料

## 6. 邏輯與規則

### 6.1 語言

- 支援：English、Tiếng Việt、日本語、繁體中文
- 變更語言將重新載入整個應用程式
- 系統類別將自動翻譯為新語言

### 6.2 貨幣

- 每種語言都有預設貨幣（例如：TWD 用於繁體中文）
- 您可以選擇不同貨幣
- 所有金額將根據選定的貨幣格式化

### 6.3 通知

- 通知僅在應用程式有權限時運作
- 通知時間取決於每個功能，可設定：
  - 定期收入/支出：`notificationTime1`、`notificationTime2`（預設 16:00 和 19:00）
  - 儲蓄帳戶和貸款：`notificationTime1`、`notificationTime2`（預設 10:00 和 19:00）
  - 特殊場合和準備步驟：根據您輸入的 `reminderTime`
  - 可以分別停用每種類型的通知（未來功能）

### 6.4 類別

- 系統類別無法刪除，只能停用
- 使用者類別可以刪除（如未使用）
- 每種類別類型都是獨立的（定期收入、日常支出等）

## 7. 重要注意事項

- **定期備份**：應定期備份資料以避免資料遺失
- **還原將覆蓋**：還原將取代所有目前資料
- **密碼**：如果您忘記密碼，可以重設（將刪除資料）
