# 財務目標

## 1. 目的

**財務目標**模組可協助您：
- 設定財務目標（例如：買房、買車）
- 規劃財務以達成目標
- 評估財務假設
- 比較貸款情境
- 追蹤目標進度

## 2. 使用時機

當您想：
- 規劃重大財務目標時
- 評估購買資產的借貸能力時
- 比較財務選項時
- 追蹤儲蓄進度時

## 3. 相關畫面

- 建立財務目標（3 個步驟）
- 目標詳細資訊和計劃
- 查看貸款計劃
- 評估假設
- 評估假設和貸款

## 4. 主要使用方式

### 4.1 建立財務目標（3 個步驟）

#### 步驟 1：輸入財務計劃

1. 前往**功能** → 選擇**計劃與假設**
2. 點擊 **➕ 新增** (FAB) 按鈕
3. 查看自動填入的資訊：
   - **平均收入**：從啟用的定期收入自動檢索（可點擊查看明細）
   - **固定支出**：從啟用的定期支出 + 貸款付款自動檢索（可點擊查看明細）
   - **目前餘額**：從目前餘額自動檢索
4. 輸入**生活支出**：每月生活支出（飲食、交通等）
5. 查看自動計算的預測：
   - 12 個月後
   - 24 個月後
   - 36 個月後
   （如果保持目前收入和支出水準）
6. 點擊**繼續**

#### 步驟 2：輸入目標資訊

1. 輸入**目標名稱**：（例如：「買房」）
2. 輸入**所需金額**：達成目標所需的總金額
3. 查看**頭期款**：從目前餘額自動填入（可編輯）
4. 點擊**繼續**

#### 步驟 3：檢查目標達成能力

1. 查看目標資訊：名稱、目標價值、頭期款、剩餘差距
2. 查看目前財務：平均收入、平均支出、平均儲蓄
3. 查看結論：
   - 「您將在大約 X 年內達成目標」（如果儲蓄 > 0）
   - 「在目前情況下，不借貸或改善財務狀況無法達成目標」（如果儲蓄 <= 0）
4. 查看下一步選項：
   - **查看貸款選項**：評估借貸能力
   - **建立收入/支出假設**：模擬財務改善
   - **結合假設 + 貸款**：最佳情境
5. 點擊**儲存目標**（可以現在儲存或稍後建立計劃）

### 4.2 查看目標清單和詳細資訊

1. 前往**功能** → 選擇**計劃與假設**
2. 查看已建立的目標清單：
   - 每個目標顯示：名稱、目標價值、已付頭期款、剩餘差距
3. （選填）使用搜尋列依名稱尋找目標
4. 點擊目標查看詳細資訊：
   - **目標資訊**：名稱、目標價值、頭期款、剩餘差距
   - **財務計劃（基準）**：點擊查看帶有平均收入、支出、儲蓄的對話框
   - **已儲存計劃清單**：已建立的貸款計劃、假設或組合

### 4.3 為目標建立貸款計劃

1. 在目標詳細資訊畫面上，點擊 **➕ 新增** 按鈕
2. 應用程式顯示選擇計劃類型的對話框，選擇**「貸款」**
3. 輸入貸款資訊：貸款金額、利率、貸款期限、計劃名稱
4. 查看自動計算的結果：每月付款、總付款金額、達成目標時間、負擔能力
5. 點擊**儲存計劃**

### 4.4 建立收入/支出假設

1. 在目標詳細資訊畫面上，點擊 **➕ 新增** 按鈕
2. 應用程式顯示選擇計劃類型的對話框，選擇**「假設」**
3. 輸入假設：
   - **增加收入**：額外金額（如無增加則留空）
   - **減少支出**：減少金額（如無減少則留空）
   - 假設名稱
4. 查看自動計算的結果：新收入、新支出、新儲蓄、達成目標的新時間
5. 點擊**儲存假設**

### 4.5 刪除財務目標

1. 前往目標詳細資訊畫面
2. 點擊目標卡片右上角的**刪除**按鈕（刪除圖示）
3. 應用程式顯示確認對話框
4. 點擊**刪除**確認

**注意**：刪除目標將刪除所有相關計劃且無法復原。

## 6. 邏輯與規則

### 6.1 預測計算

- 預測基於：
  - 收入 - 固定支出 - 生活支出 = 儲蓄/月
  - 目前餘額 + (儲蓄/月 × 月數)

### 6.2 目標

- 剩餘差距 = 所需金額 - 頭期款
- 估計時間 = 剩餘差距 / 平均儲蓄（月）
- 如果平均儲蓄 <= 0：不借貸或改善財務狀況無法達成目標

### 6.3 貸款計劃

- 計算基於：
  - 貸款金額
  - 利率
  - 期限
  - 自動建立付款時間表

### 6.4 財務假設

- 評估假設，例如：
  - 增加/減少收入
  - 增加/減少支出
  - 變更利率
- 查看對目標達成能力的影響

## 7. 重要注意事項

- **需要高級模組**：此功能僅供高級使用者使用
- **預測僅供參考**：基於收入和支出穩定的假設
- **可建立多個計劃**：您可以建立多個計劃（貸款、假設、組合）進行比較
- **基準已儲存**：建立目標時會儲存初始財務計劃（基準），用於與後續計劃比較
- **自動計算**：收入和固定支出會從啟用的定期項目自動檢索，包括銀行貸款付款
- **刪除目標**：刪除目標時，所有相關計劃也會被刪除且無法還原
