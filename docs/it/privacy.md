## 2. Dati Raccolti

### L'Applicazione NON Raccoglie

L'applicazione **non raccoglie** le seguenti informazioni:

- Nome reale
- Email (eccetto quando l'utente contatta attivamente il supporto)
- Numero di telefono
- Identificatori personali
- Dati finanziari reali (informazioni bancarie, carte di credito)

> **Principio:** L'applicazione opera con un modello **local-first**, il che significa che i dati sono memorizzati sul tuo dispositivo, non inviati a un server.

## 4. Backup e Condivisione Dati

### Backup

- Il backup viene **eseguito attivamente** dall'utente
- I file di backup vengono salvati dove **l'utente sceglie** (dispositivo, cloud personale come iCloud, Google Drive)
- L'applicazione non esegue automaticamente il backup dei dati nel cloud

### Condivisione Dati

- La condivisione dei dati avviene solo quando l'utente **esporta o condivide attivamente** file di backup
- L'applicazione non condivide automaticamente i dati con terze parti
