# 额外收入

## 1. 目的

**额外收入**模块可帮助您记录非定期、无固定周期的收入，例如：
- 线上销售
- 自由工作
- 奖金
- 现金礼物
- 其他不规则收入

与**定期收入**不同，额外收入没有自动周期，您必须每次手动输入。

## 2. 使用时机

当您想：
- 记录随机、非定期收入时
- 追踪某段时间的总收入时
- 分析额外收入趋势时
- 计算到每月预算时

## 3. 相关画面

- 额外收入清单
- 新增收入
- 编辑收入

## 4. 主要使用方式

### 4.1 新增额外收入

1. 前往**功能** → 选择**额外收入**
2. 点击右下角的 **+** (FAB) 按钮
3. 填写信息：
   - **类别**：选择或创建新类别
   - **金额**：输入收到的金额
   - **日期**：选择收到钱的日期（默认为今天）
   - **备注**：详细说明（选填）
4. 点击**保存**

### 4.2 查看收入清单

1. 前往**功能** → 选择**额外收入**
2. 清单根据您设定的版面配置显示（1、2、3 或 4 栏）
3. 使用**搜索**依类别或备注筛选
4. 选择**时间筛选**：今天 / 本周 / 本月 / 上月 / 自定义

### 4.3 编辑收入

1. 前往额外收入清单
2. 长按要编辑的项目
3. 从菜单中选择**编辑**
4. 更新信息
5. 点击**保存**

### 4.4 删除收入

1. 前往额外收入清单
2. 长按要删除的项目
3. 从菜单中选择**删除**
4. 确认删除

## 6. 逻辑与规则

### 6.1 显示版面

- 您可以设定栏数：1、2、3 或 4 栏
- 版面配置保存在设置中，适用于所有额外收入清单

### 6.2 时间筛选

- **今天**：仅显示今天的收入
- **本周**：从周开始到今天
- **本月**：从月初到今天
- **上月**：整个上个月
- **自定义**：选择自定义时间范围

### 6.3 搜索

- 在**类别名称**和**备注**中搜索
- 不区分大小写
- 输入时即时搜索

### 6.4 预算整合

- 额外收入会计算到预算中的"额外收入"
- 帮助您追踪每月总收入

## 7. 重要注意事项

- **无周期**：额外收入没有自动周期，您必须每次手动输入
- **可删除**：您可以删除任何收入
- **预算整合**：额外收入会自动计算到当月预算中
