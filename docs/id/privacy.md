# Kebijakan Privasi

## 1. Pengenalan

**Daily Life Assistant** berkomitmen untuk menghormati privasi pengguna. Kebijakan privasi ini menjelaskan cara kami menangani informasi Anda saat menggunakan aplikasi.

Kebijakan ini berlaku untuk:

- Aplikasi Daily Life Assistant
- Semua platform (iOS, Android)

---

## 2. Data yang Dikumpulkan

### Aplikasi TIDAK Mengumpulkan

Aplikasi **tidak mengumpulkan** informasi berikut:

- Nama asli
- Email (kecuali pengguna secara aktif menghubungi dukungan)
- Nomor telepon
- Pengidentifikasi pribadi
- Data keuangan aktual (informasi bank, kartu kredit)

> **Prinsip:** Aplikasi beroperasi dengan model **local-first**, artinya data disimpan di perangkat Anda, tidak dikirim ke server.

---

## 3. Data yang Dimasukkan Pengguna

### Jenis Data

Saat menggunakan aplikasi, Anda dapat memasukkan data seperti:

- Pengeluaran (pendapatan, pengeluaran harian, pengeluaran berulang)
- Tugas (daftar tugas, acara khusus)
- Catatan pribadi
- Rencana (anggaran, tujuan keuangan)

## 4. Cadangan & Berbagi Data

### Cadangan

- Cadangan **dilakukan secara aktif** oleh pengguna
- File cadangan disimpan di tempat yang **dipilih pengguna** (perangkat, cloud pribadi seperti iCloud, Google Drive)
- Aplikasi tidak secara otomatis mencadangkan data ke cloud

### Berbagi Data

- Berbagi data hanya terjadi ketika pengguna **secara aktif mengekspor atau berbagi** file cadangan
- Aplikasi tidak secara otomatis berbagi data dengan pihak ketiga

---

## 5. Data Pembayaran

### Pembayaran Premium

- Pembayaran untuk paket Premium diproses oleh:
  - **App Store** (iOS)
  - **Google Play** (Android)

### Informasi Pembayaran

Aplikasi:

- **Tidak menyimpan** informasi kartu kredit
- **Tidak mengakses** informasi pembayaran Anda
- Hanya menerima konfirmasi tentang status Premium dari App Store/Google Play

---

## 6. Analitik & Pelacakan

### Iklan

Aplikasi:

- **Tidak menggunakan** iklan
- **Tidak menggunakan** pelacakan iklan
- **Tidak berbagi** data dengan pengiklan

### Analitik

- Aplikasi dapat menggunakan analitik anonim (jika ada)
- Analitik **tidak mengandung data pribadi** pengguna
- Saat ini, aplikasi **tidak menggunakan analitik**

---

## 7. Hak Pengguna

Anda berhak untuk:

### Melihat Data

- Melihat semua data yang telah Anda masukkan dalam aplikasi
- Mengekspor data sebagai file cadangan

### Menghapus Data

Anda dapat menghapus data dengan:

- Menghapus setiap item secara manual dalam aplikasi
- Mencopot pemasangan aplikasi (akan menghapus semua data di perangkat)

### Mengontrol Cadangan

- Anda memiliki kontrol penuh atas cadangan data
- Anda memutuskan kapan dan di mana menyimpan file cadangan

---

## 8. Keamanan Data

### Tindakan Keamanan

- Data dilindungi oleh **enkripsi sistem operasi** (iOS/Android)
- Data disimpan dalam **basis data aman** di perangkat Anda
- Keamanan bergantung pada **keamanan perangkat** pengguna (kata sandi, Face ID, sidik jari)

### Batasan Keamanan

Aplikasi tidak dapat melindungi data jika:

- Perangkat dikompromikan atau hilang
- Pengguna berbagi file cadangan dengan orang lain
- Perangkat di-jailbreak (iOS) atau di-root (Android)

> **Rekomendasi:** Selalu lindungi perangkat Anda dengan kata sandi yang kuat dan tindakan keamanan lainnya.

---

## 9. Anak-anak

- Aplikasi **tidak ditujukan untuk** anak-anak di bawah 13 tahun
- Aplikasi **tidak secara sengaja mengumpulkan** data dari anak-anak
- Jika Anda adalah orang tua dan menemukan bahwa anak Anda telah memberikan informasi pribadi, silakan hubungi kami

---

## 10. Perubahan Kebijakan

### Pembaruan Kebijakan

- Kebijakan privasi dapat diperbarui dari waktu ke waktu
- Versi baru kebijakan akan dipublikasikan di **situs web** (GitHub Pages)
- **Tanggal efektif** versi baru akan dinyatakan dengan jelas

### Pemberitahuan Perubahan

- Kami akan berusaha memberi tahu tentang perubahan penting
- Penggunaan aplikasi yang berkelanjutan setelah kebijakan diperbarui dipahami sebagai **persetujuan dengan kebijakan baru**

---

## 11. Kontak

### Cara Menghubungi

Jika Anda memiliki pertanyaan tentang kebijakan privasi ini, silakan hubungi melalui:

- **Email dukungan** (lihat di bagian Pengaturan aplikasi)

### Keamanan Informasi saat Menghubungi

- Aplikasi **tidak memerlukan** Anda untuk memberikan data sensitif saat menghubungi dukungan
- Berikan hanya informasi yang diperlukan untuk menyelesaikan masalah Anda

---

## Ringkasan

Kebijakan privasi ini bertujuan untuk:

- **Transparansi** tentang cara aplikasi menangani data Anda
- **Melindungi** hak privasi pengguna
- **Mematuhi** persyaratan Apple App Privacy dan Google Data Safety

**Prinsip Inti:** Data Anda adalah milik Anda. Aplikasi hanyalah alat bagi Anda untuk mengelola data di perangkat Anda.
