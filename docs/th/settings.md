# การตั้งค่า

## 1. วัตถุประสงค์

โมดูล **การตั้งค่า** ช่วยให้คุณกำหนดค่าแอปตามความต้องการส่วนบุคคล รวมถึง:
- ภาษาและสกุลเงิน
- การแจ้งเตือน
- หมวดหมู่
- สำรองข้อมูลและกู้คืนข้อมูล
- ความปลอดภัย (รหัสผ่าน, Face ID)

## 2. เมื่อใดควรใช้

ใช้โมดูลนี้เมื่อคุณต้องการ:
- เปลี่ยนภาษาหรือสกุลเงิน
- เปิด/ปิดการแจ้งเตือน
- จัดการหมวดหมู่ (เพิ่ม, แก้ไข, ลบ, ตั้งค่าเริ่มต้น)
- สำรองหรือกู้คืนข้อมูล
- เปลี่ยนรหัสผ่านหรือเปิดใช้งาน Face ID

## 3. หน้าจอที่เกี่ยวข้อง

- หน้าจอการตั้งค่าหลัก
- การตั้งค่าพื้นฐาน
- เลือกภาษา
- เลือกสกุลเงิน
- จัดการหมวดหมู่
- สำรองข้อมูล
- กู้คืนข้อมูล
- เปลี่ยนรหัสผ่าน
- เปิดใช้งาน Face ID / ลายนิ้วมือ

## 4. การใช้งานหลัก

### 4.1 เปลี่ยนภาษา

1. ไปที่ **การตั้งค่า** → **การแสดงผลและภาษา** → **ภาษา**
2. เลือกภาษาที่ต้องการ
3. แอปจะโหลดใหม่ด้วยภาษาที่เลือกอัตโนมัติ

### 4.2 เปลี่ยนสกุลเงิน

1. ไปที่ **การตั้งค่า** → **การแสดงผลและภาษา** → **สกุลเงิน**
2. เลือกประเภทสกุลเงิน
3. จำนวนเงินทั้งหมดจะแสดงในหน่วยใหม่

### 4.3 เปิด/ปิดการแจ้งเตือน

1. ไปที่ **การตั้งค่า** → **การแจ้งเตือน**
2. เปิด/ปิดสวิตช์ **การแจ้งเตือน**
3. ถ้าเปิดใช้งาน คุณจะได้รับการแจ้งเตือนเกี่ยวกับ:
   - รายได้ประจำที่ครบกำหนด
   - ค่าใช้จ่ายประจำที่ครบกำหนด
   - บัญชีเงินออมที่ครบกำหนด
   - โอกาสพิเศษที่กำลังจะมาถึง

### 4.4 จัดการหมวดหมู่

1. ไปที่ **การตั้งค่า** → **หมวดหมู่**
2. เลือกประเภทหมวดหมู่ที่ต้องการจัดการ:
   - รายได้ประจำ
   - รายได้พิเศษ
   - ค่าใช้จ่ายประจำ
   - ค่าใช้จ่ายรายวัน
3. เพิ่ม/แก้ไข/ลบหมวดหมู่
4. ตั้งหมวดหมู่เริ่มต้น (สำหรับค่าใช้จ่ายรายวัน)

### 4.5 สำรองข้อมูล

1. ไปที่ **การตั้งค่า** → **สำรองข้อมูลและข้อมูล** → **สำรองข้อมูล**
2. เลือกตำแหน่งบันทึก (ระบบไฟล์)
3. แตะ **สำรองข้อมูล**
4. ไฟล์สำรองข้อมูลจะถูกสร้าง

### 4.6 กู้คืนข้อมูล

1. ไปที่ **การตั้งค่า** → **สำรองข้อมูลและข้อมูล** → **กู้คืนข้อมูล**
2. เลือกไฟล์สำรองข้อมูลจากระบบไฟล์
3. ยืนยันการกู้คืน
4. **หมายเหตุ**: การกู้คืนจะเขียนทับข้อมูลปัจจุบัน

## 6. ตรรกะและกฎ

### 6.1 ภาษา

- รองรับ: ภาษาอังกฤษ, ภาษาไทย, 日本語
- การเปลี่ยนภาษาจะโหลดแอปทั้งหมดใหม่
- หมวดหมู่ระบบจะแปลเป็นภาษาที่เลือกอัตโนมัติ

### 6.2 สกุลเงิน

- แต่ละภาษามีสกุลเงินเริ่มต้น (เช่น THB สำหรับภาษาไทย)
- คุณสามารถเลือกสกุลเงินอื่นได้
- จำนวนเงินทั้งหมดจะถูกจัดรูปแบบตามสกุลเงินที่เลือก

### 6.3 การแจ้งเตือน

- การแจ้งเตือนจะทำงานเมื่อแอปได้รับอนุญาตเท่านั้น
- เวลาแจ้งเตือนขึ้นอยู่กับแต่ละฟังก์ชันและสามารถกำหนดได้:
  - รายได้/ค่าใช้จ่ายประจำ: `notificationTime1`, `notificationTime2` (ค่าเริ่มต้น 16:00 และ 19:00)
  - บัญชีเงินออมและเงินกู้: `notificationTime1`, `notificationTime2` (ค่าเริ่มต้น 10:00 และ 19:00)
  - โอกาสพิเศษและขั้นตอนการเตรียมตัว: ตาม `reminderTime` ที่คุณกรอก
  - สามารถปิดการแจ้งเตือนสำหรับแต่ละประเภทแยกกันได้ (ในอนาคต)

### 6.4 หมวดหมู่

- หมวดหมู่ระบบไม่สามารถลบได้ สามารถปิดการใช้งานเท่านั้น
- หมวดหมู่ผู้ใช้สามารถลบได้ (ถ้าไม่ได้ใช้งาน)
- แต่ละประเภทหมวดหมู่เป็นอิสระ (รายได้ประจำ, ค่าใช้จ่ายรายวัน, ฯลฯ)

## 7. หมายเหตุสำคัญ

- **สำรองข้อมูลเป็นประจำ**: ควรสำรองข้อมูลเป็นระยะเพื่อหลีกเลี่ยงการสูญหายของข้อมูล
- **การกู้คืนจะเขียนทับ**: การกู้คืนจะแทนที่ข้อมูลปัจจุบันทั้งหมด
- **รหัสผ่าน**: ถ้าคุณลืมรหัสผ่าน คุณสามารถรีเซ็ตได้ (จะลบข้อมูล)
