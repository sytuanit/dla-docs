# 설정

## 1. 목적

**설정** 모듈에서는 개인적인 필요에 따라 앱을 설정할 수 있습니다:
- 언어 및 통화
- 알림
- 카테고리
- 데이터 백업 및 복원
- 보안 (비밀번호, Face ID)

## 2. 사용 시기

다음과 같은 경우 이 모듈을 사용합니다:
- 언어 또는 통화 변경
- 알림 활성화/비활성화
- 카테고리 관리 (추가, 편집, 삭제, 기본값 설정)
- 데이터 백업 또는 복원
- 비밀번호 변경 또는 Face ID 활성화

## 3. 관련 화면

- 메인 설정 화면
- 기본 설정
- 언어 선택
- 통화 선택
- 카테고리 관리
- 데이터 백업
- 데이터 복원
- 비밀번호 변경
- Face ID / 지문 인증 활성화

## 4. 주요 사용 방법

### 4.1 언어 변경

1. **설정** → **표시 및 언어** → **언어**로 이동
2. 원하는 언어 선택
3. 앱이 새 언어로 자동으로 다시 로드됩니다

### 4.2 통화 변경

1. **설정** → **표시 및 언어** → **통화**로 이동
2. 통화 유형 선택
3. 모든 금액이 새 단위로 표시됩니다

### 4.3 알림 활성화/비활성화

1. **설정** → **알림**으로 이동
2. **알림** 스위치 전환
3. 활성화하면 다음 알림을 수신합니다:
   - 정기 수입 만기일
   - 정기 지출 만기일
   - 저축 계좌 만기
   - 특별한 날 도래

### 4.4 카테고리 관리

1. **설정** → **카테고리**로 이동
2. 관리할 카테고리 유형 선택:
   - 정기 수입
   - 임시 수입
   - 정기 지출
   - 일일 지출
3. 카테고리 추가/편집/삭제
4. 기본 카테고리 설정 (일일 지출용)

### 4.5 데이터 백업

1. **설정** → **백업 및 데이터** → **백업**으로 이동
2. 저장 위치 선택 (파일 시스템)
3. **백업** 탭
4. 백업 파일이 생성됩니다

### 4.6 데이터 복원

1. **설정** → **백업 및 데이터** → **복원**으로 이동
2. 파일 시스템에서 백업 파일 선택
3. 복원 확인
4. **참고**: 복원은 현재 데이터를 덮어씁니다

## 6. 로직 및 규칙

### 6.1 언어

- 지원 언어: 영어, Tiếng Việt, 한국어
- 언어를 변경하면 앱 전체가 다시 로드됩니다
- 시스템 카테고리는 새 언어로 자동 번역됩니다

### 6.2 통화

- 각 언어에는 기본 통화가 있습니다 (예: 영어의 경우 USD)
- 다른 통화를 선택할 수 있습니다
- 모든 금액이 선택한 통화에 따라 포맷됩니다

### 6.3 알림

- 알림은 앱에 권한이 있는 경우에만 작동합니다
- 알림 시간은 각 기능에 따라 다르며 설정할 수 있습니다:
  - 정기 수입/지출: `notificationTime1`, `notificationTime2` (기본값 16:00 및 19:00)
  - 저축 계좌 및 대출: `notificationTime1`, `notificationTime2` (기본값 10:00 및 19:00)
  - 특별한 날 및 준비 단계: 입력한 `reminderTime`에 따름
  - 각 유형의 알림을 개별적으로 비활성화할 수 있습니다 (향후)

### 6.4 카테고리

- 시스템 카테고리는 삭제할 수 없으며 비활성화만 가능
- 사용자 카테고리는 삭제할 수 있습니다 (사용되지 않는 경우)
- 각 카테고리 유형은 독립적입니다 (정기 수입, 일일 지출 등)

## 7. 중요 참고 사항

- **정기적으로 백업**: 데이터 손실을 피하기 위해 정기적으로 데이터를 백업해야 합니다
- **복원은 덮어쓰기**: 복원은 모든 현재 데이터를 대체합니다
- **비밀번호**: 비밀번호를 잊은 경우 재설정할 수 있습니다 (데이터가 삭제됩니다)
