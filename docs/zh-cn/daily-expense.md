# 日常支出

## 1. 目的

**日常支出**模块可帮助您记录常规、非固定支出，例如：
- 饮食
- 购物
- 交通
- 娱乐
- 其他灵活支出

与**定期支出**不同，日常支出通常在金额和频率上变化，没有固定周期。

## 2. 使用时机

当您想：
- 记录随机、非定期支出时
- 追踪日常开支以控制预算时
- 按类别分析支出趋势时
- 查看某段时间的总支出时

## 3. 相关画面

- 日常支出清单
- 新增支出
- 编辑支出

## 4. 主要使用方式

### 4.1 新增日常支出

1. 前往**功能** → 选择**日常支出**
2. 点击右下角的 **+** (FAB) 按钮
3. 填写信息：
   - **类别**：选择类别（或使用已设定的默认类别）
   - **金额**：输入支出金额
   - **日期**：选择支出日期（默认为今天）
   - **备注**：详细说明（选填）
4. 点击**保存**

### 4.2 查看支出清单

1. 前往**功能** → 选择**日常支出**
2. 清单根据您设定的版面配置显示（2、3 或 4 栏）
3. 使用**搜索**依类别或备注筛选
4. 选择**时间筛选**：今天 / 本周 / 本月 / 上月 / 自定义

### 4.3 编辑支出

1. 前往日常支出清单
2. 长按要编辑的项目
3. 从菜单中选择**编辑**
4. 更新信息
5. 点击**保存**

### 4.4 删除支出

1. 前往日常支出清单
2. 长按要删除的项目
3. 从菜单中选择**删除**
4. 确认删除

### 4.5 设定默认类别

1. 前往**设置** → **类别** → **日常支出类别**
2. 点击要设为默认的类别
3. 选择**设为默认**
4. 新增支出时，此类别将自动选中

## 6. 逻辑与规则

### 6.1 显示版面

- 您可以设定栏数：2、3 或 4 栏
- 版面配置保存在设置中，适用于所有支出清单

### 6.2 时间筛选

- **今天**：仅显示今天的支出
- **本周**：从周开始到今天
- **本月**：从月初到今天
- **上月**：整个上个月
- **自定义**：选择自定义时间范围

### 6.3 搜索

- 在**类别名称**和**备注**中搜索
- 不区分大小写
- 输入时即时搜索

### 6.4 默认类别

- 如果您设定了默认类别，开启新增画面时，该类别将自动选中
- 备注也可以根据类别自动填入（如已设定）

### 6.5 总支出

- 总支出根据目前选定的时间筛选计算
- 显示在清单底部

## 7. 重要注意事项

- **无周期**：日常支出没有自动周期，您必须每次手动输入
- **可删除**：您可以删除任何支出（与定期支出不同）
- **无预算整合**：日常支出不会自动计算到预算中（您必须自行追踪）
- **自定义类别**：您可以在设置中创建新类别
