# 일일 지출

## 1. 목적

**일일 지출** 모듈은 다음과 같은 정기적이지 않고 고정되지 않은 지출을 기록하는 데 도움이 됩니다:
- 식사·외식
- 쇼핑
- 교통비
- 오락
- 기타 유연한 지출

**정기 지출**과 달리, 일일 지출은 금액과 빈도가 변동하는 경우가 많으며 고정 주기가 없습니다.

## 2. 사용 시기

다음과 같은 경우 이 모듈을 사용합니다:
- 무작위이고 비정기적인 지출 기록
- 일일 지출 추적 및 예산 관리
- 카테고리별 지출 추세 분석
- 기간 내 총 지출 표시

## 3. 관련 화면

- 일일 지출 목록
- 새 지출 추가
- 지출 편집

## 4. 주요 사용 방법

### 4.1 일일 지출 추가

1. **기능** → **일일 지출** 선택
2. 오른쪽 하단의 **+** (FAB) 버튼 탭
3. 정보 입력:
   - **카테고리**: 카테고리 선택 (또는 설정된 경우 기본 카테고리 사용)
   - **금액**: 지출액 입력
   - **날짜**: 지출일 선택 (기본값은 오늘)
   - **메모**: 상세 설명 (선택 사항)
4. **저장** 탭

### 4.2 지출 목록 표시

1. **기능** → **일일 지출** 선택
2. 목록이 설정된 레이아웃(2, 3 또는 4열)에 따라 표시됩니다
3. **검색**을 사용하여 카테고리 또는 메모로 필터링
4. **시간 필터** 선택: 오늘 / 이번 주 / 이번 달 / 지난 달 / 사용자 지정

### 4.3 지출 편집

1. 일일 지출 목록으로 이동
2. 편집할 항목 길게 누르기
3. 메뉴에서 **편집** 선택
4. 정보 업데이트
5. **저장** 탭

### 4.4 지출 삭제

1. 일일 지출 목록으로 이동
2. 삭제할 항목 길게 누르기
3. 메뉴에서 **삭제** 선택
4. 삭제 확인

### 4.5 기본 카테고리 설정

1. **설정** → **카테고리** → **일일 지출 카테고리**로 이동
2. 기본으로 설정할 카테고리 탭
3. **기본으로 설정** 선택
4. 새 지출을 추가할 때 이 카테고리가 자동으로 선택됩니다

## 6. 로직 및 규칙

### 6.1 표시 레이아웃

- 열 수를 설정할 수 있습니다: 2, 3 또는 4열
- 레이아웃은 설정에 저장되며 모든 지출 목록에 적용됩니다

### 6.2 시간 필터

- **오늘**: 오늘의 지출만 표시
- **이번 주**: 주 시작부터 오늘까지
- **이번 달**: 달 시작부터 오늘까지
- **지난 달**: 전월 전체
- **사용자 지정**: 사용자 지정 시간 범위 선택

### 6.3 검색

- **카테고리명**과 **메모**로 검색
- 대소문자 구분 없음
- 입력에 따라 실시간 검색

### 6.4 기본 카테고리

- 기본 카테고리를 설정한 경우, 추가 화면을 열면 해당 카테고리가 자동으로 선택됩니다
- 메모도 카테고리에 따라 자동 입력할 수 있습니다(설정된 경우)

### 6.5 총 지출

- 총 지출은 현재 선택된 시간 필터에 따라 계산됩니다
- 목록 하단에 표시됩니다

## 7. 중요 참고 사항

- **주기 없음**: 일일 지출에는 자동 주기가 없으며 매번 수동으로 입력해야 합니다
- **삭제 가능**: 모든 지출을 삭제할 수 있습니다(정기 지출과 다름)
- **예산 통합 없음**: 일일 지출은 예산에 자동 계산되지 않습니다(직접 추적해야 함)
- **사용자 지정 카테고리**: 설정에서 새 카테고리를 만들 수 있습니다
