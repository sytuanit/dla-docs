# 日常支出

## 1. 目的

**日常支出**模組可協助您記錄常規、非固定支出，例如：
- 飲食
- 購物
- 交通
- 娛樂
- 其他彈性支出

與**定期支出**不同，日常支出通常在金額和頻率上變化，沒有固定週期。

## 2. 使用時機

當您想：
- 記錄隨機、非定期支出時
- 追蹤日常開支以控制預算時
- 按類別分析支出趨勢時
- 查看某段時間的總支出時

## 3. 相關畫面

- 日常支出清單
- 新增支出
- 編輯支出

## 4. 主要使用方式

### 4.1 新增日常支出

1. 前往**功能** → 選擇**日常支出**
2. 點擊右下角的 **+** (FAB) 按鈕
3. 填寫資訊：
   - **類別**：選擇類別（或使用已設定的預設類別）
   - **金額**：輸入支出金額
   - **日期**：選擇支出日期（預設為今天）
   - **備註**：詳細說明（選填）
4. 點擊**儲存**

### 4.2 查看支出清單

1. 前往**功能** → 選擇**日常支出**
2. 清單根據您設定的版面配置顯示（2、3 或 4 欄）
3. 使用**搜尋**依類別或備註篩選
4. 選擇**時間篩選**：今天 / 本週 / 本月 / 上月 / 自訂

### 4.3 編輯支出

1. 前往日常支出清單
2. 長按要編輯的項目
3. 從選單中選擇**編輯**
4. 更新資訊
5. 點擊**儲存**

### 4.4 刪除支出

1. 前往日常支出清單
2. 長按要刪除的項目
3. 從選單中選擇**刪除**
4. 確認刪除

### 4.5 設定預設類別

1. 前往**設定** → **類別** → **日常支出類別**
2. 點擊要設為預設的類別
3. 選擇**設為預設**
4. 新增支出時，此類別將自動選中

## 6. 邏輯與規則

### 6.1 顯示版面

- 您可以設定欄數：2、3 或 4 欄
- 版面配置儲存在設定中，適用於所有支出清單

### 6.2 時間篩選

- **今天**：僅顯示今天的支出
- **本週**：從週開始到今天
- **本月**：從月初到今天
- **上月**：整個上個月
- **自訂**：選擇自訂時間範圍

### 6.3 搜尋

- 在**類別名稱**和**備註**中搜尋
- 不區分大小寫
- 輸入時即時搜尋

### 6.4 預設類別

- 如果您設定了預設類別，開啟新增畫面時，該類別將自動選中
- 備註也可以根據類別自動填入（如已設定）

### 6.5 總支出

- 總支出根據目前選定的時間篩選計算
- 顯示在清單底部

## 7. 重要注意事項

- **無週期**：日常支出沒有自動週期，您必須每次手動輸入
- **可刪除**：您可以刪除任何支出（與定期支出不同）
- **無預算整合**：日常支出不會自動計算到預算中（您必須自行追蹤）
- **自訂類別**：您可以在設定中建立新類別
