# ວັນພິເສດ

## 1. ຈຸດປະສົງ

ໂມດູນ **ວັນພິເສດ** ຊ່ວຍທ່ານຈັດການວັນພິເສດຕະຫຼອດປີແລະກະກຽມສໍາລັບພວກມັນ, ລວມທັງ:
- ຈັດການວັນພິເສດ (ວັນເດືອນເກີດ, ວັນພັກ, ແລະອື່ນໆ)
- ສ້າງລາຍການວຽກທີ່ຕ້ອງເຮັດ (ຂັ້ນຕອນການກະກຽມ)
- ແກ້ວລາຍການກວດສອບກັບແຕ່ລະຂັ້ນຕອນການກະກຽມ
- ການແຈ້ງເຕືອນກ່ອນວັນພິເສດ
- ຕິດຕາມຄວາມຄືບໜ້າການກະກຽມ

## 2. ເມື່ອໃດທີ່ຄວນໃຊ້

ໃຊ້ໂມດູນນີ້ເມື່ອທ່ານຕ້ອງການ:
- ຈັດການວັນພິເສດຕະຫຼອດປີ
- ກະກຽມສໍາລັບວັນພິເສດສໍາຄັນ
- ສ້າງລາຍການວຽກທີ່ຕ້ອງເຮັດ
- ໄດ້ຮັບການແຈ້ງເຕືອນກ່ອນວັນພິເສດ

## 3. ຫນ້າຈໍທີ່ກ່ຽວຂ້ອງ

- ລາຍການວັນພິເສດ
- ເພີ່ມວັນພິເສດໃໝ່
- ລາຍລະອຽດວັນພິເສດແລະຂັ້ນຕອນການກະກຽມ
- ເພີ່ມຂັ້ນຕອນການກະກຽມ
- ເລືອກລາຍການກວດສອບ
- ສ້າງລາຍການກວດສອບໃໝ່

## 4. ການນໍາໃຊ້ຫຼັກ

### 4.1 ເພີ່ມວັນພິເສດ

1. ໄປທີ່ **ຟັງຊັນ** → ເລືອກ **ວັນພິເສດ**
2. ແຕະປຸ່ມ **+** (FAB)
3. ຕື່ມຂໍ້ມູນ:
   - **ຊື່ວັນພິເສດ**: (ຕົວຢ່າງ: "ວັນເດືອນເກີດຂອງແມ່")
   - **ວັນທີ**: ເລືອກວັນ/ເດືອນ (DatePicker ເລືອກພຽງແຕ່ວັນ/ເດືອນ, ບໍ່ມີປີ)
   - **ໃຊ້ປະຕິທິນຈັນທະຄະຕິ**: (ທາງເລືອກ) ເລືອກຖ້າທ່ານຕ້ອງການໃຊ້ປະຕິທິນຈັນທະຄະຕິ
     - ຖ້າເລືອກ: ໃສ່ວັນຈັນທະຄະຕິແລະເດືອນ, ແອັບຄິດໄລ່ວັນທີສຸລິນທີ່ໃກ້ທີ່ສຸດອັດຕະໂນມັດ
   - **ຊໍ້າຊ້ອນ**: ລາຍປີ / ປີນີ້ເທົ່ານັ້ນ
   - **ສະແດງການແຈ້ງເຕືອນທີ່**: ເລືອກເວລາ (ຈໍາເປັນ, ຕົວຢ່າງ: 07:00)
   - **ໝາຍເຫດ**: ຂໍ້ມູນເພີ່ມເຕີມ (ທາງເລືອກ)
4. (ທາງເລືອກ) ເພີ່ມຂັ້ນຕອນການກະກຽມ (ເບິ່ງ 4.2)
5. ແຕະ **ບັນທຶກ**

### 4.2 ເພີ່ມຂັ້ນຕອນການກະກຽມ

1. ເມື່ອເພີ່ມວັນພິເສດໃໝ່: ແຕະ **+ ເພີ່ມຂັ້ນຕອນ** ໃນສ່ວນ "ຂັ້ນຕອນການກະກຽມ"
2. ຫຼືຈາກລາຍລະອຽດວັນພິເສດ: ແຕະ **+ ເພີ່ມຂັ້ນຕອນ**
3. ຕື່ມຂໍ້ມູນ:
   - **ເມື່ອໃດ?**: "X ວັນກ່ອນ" ຫຼື "ໃນມື້ນັ້ນ"
   - **ຈໍານວນວັນ**: (ຖ້າເລືອກ "X ວັນກ່ອນ") ໃສ່ຈໍານວນວັນກ່ອນວັນພິເສດ
   - **ສະແດງການແຈ້ງເຕືອນທີ່**: ເລືອກເວລາ (ຈໍາເປັນ)
   - **ຊໍ້າຊ້ອນລາຍວັນຈົນກວ່າຈະສໍາເລັດ**: (ທາງເລືອກ) ເລືອກຖ້າທ່ານຕ້ອງການການແຈ້ງເຕືອນລາຍວັນ
   - **ເນື້ອຫາ**: ຊື່ຂັ້ນຕອນ (ຈໍາເປັນ, ຕົວຢ່າງ: "ຊື້ຂອງຂວັນ")
   - **ໝາຍເຫດ**: (ທາງເລືອກ)
   - **ໃຊ້ລາຍການກວດສອບ**: (ທາງເລືອກ) ເລືອກເພື່ອເຊື່ອມກັບລາຍການກວດສອບການຊື້
4. ແຕະ **ເພີ່ມ** (ຫຼື FAB "ນໍາໃຊ້")

### 4.3 ສ້າງລາຍການກວດສອບ

1. ເມື່ອເພີ່ມຂັ້ນຕອນການກະກຽມ, ເລືອກ **ໃຊ້ລາຍການກວດສອບ**
2. ຫນ້າຈໍ "ເລືອກລາຍການກວດສອບການຊື້" ອັດຕະໂນມັດເປີດ
3. ແຕະ FAB **+** ເພື່ອສ້າງລາຍການກວດສອບໃໝ່
4. ໃສ່ຊື່ລາຍການກວດສອບ
5. ເພີ່ມລາຍການ:
   - ໃສ່ຊື່ລາຍການ
   - ແຕະ **+** ເພື່ອເພີ່ມລາຍການໃໝ່
6. ແຕະ **ບັນທຶກ**
7. ລາຍການກວດສອບໃໝ່ຖືກເລືອກອັດຕະໂນມັດແລະກັບໄປຫນ້າຈໍ "ເພີ່ມຂັ້ນຕອນການກະກຽມ"

### 4.4 ກໍາຫນົດຂັ້ນຕອນເປັນສໍາເລັດ

1. ໄປທີ່ລາຍລະອຽດວັນພິເສດ
2. ຊອກຫາຂັ້ນຕອນທີ່ຈະກໍາຫນົດ
3. ແຕະ checkbox [ ] ເພື່ອປ່ຽນເປັນ [✓]
4. ຖ້າມີລາຍການກວດສອບ, ແຕະຊື່ລາຍການກວດສອບເພື່ອເບິ່ງແລະເລືອກ/ຍົກເລີກລາຍການ

### 4.5 ເບິ່ງຄວາມຄືບໜ້າ

1. ໄປທີ່ລາຍລະອຽດວັນພິເສດ
2. ເບິ່ງສ່ວນ "ສະຫຼຸບ":
   - ຂັ້ນຕອນການກະກຽມ: ຈໍານວນຂັ້ນຕອນທັງໝົດ
   - ສໍາເລັດ: ຈໍານວນຂັ້ນຕອນທີ່ເລືອກ / ຂັ້ນຕອນທັງໝົດ
   - ສະຖານະ: ບໍ່ເລີ່ມຕົ້ນ / ກໍາລັງດໍາເນີນຢູ່ / ສໍາເລັດ

### 4.6 ແກ້ໄຂວັນພິເສດ

1. ໄປທີ່ລາຍລະອຽດວັນພິເສດ
2. ແຕະ hyperlink **ແກ້ໄຂ ›** ໃນ header
3. ແກ້ໄຂຂໍ້ມູນ: ຊື່, ວັນທີ, ຊໍ້າຊ້ອນ, ເວລາແຈ້ງເຕືອນ, ໝາຍເຫດ
4. ແຕະ **ບັນທຶກ**

### 4.7 ແກ້ໄຂຂັ້ນຕອນການກະກຽມ

1. ໄປທີ່ລາຍລະອຽດວັນພິເສດ
2. ແຕະທີ່ຂັ້ນຕອນເພື່ອແກ້ໄຂ (ກົດທີ່ລາຍການທັງໝົດ, ນອກຈາກໄອຄອນລຶບ)
3. ແກ້ໄຂຂໍ້ມູນ: ເວລາ, ເນື້ອຫາ, ລາຍການກວດສອບ
4. ແຕະ **ນໍາໃຊ້** (ຫຼື FAB)

## 5. ຕົວຢ່າງ & ການສະແດງຜົນ UI

### OCCASION-01: ສ້າງວັນພິເສດໃໝ່ (ວັນເດືອນເກີດດ້ວຍຂັ້ນຕອນການກະກຽມ)

**ເປົ້າຫມາຍ**: ສ້າງວັນພິເສດໃໝ່ (ວັນເດືອນເກີດ) ດ້ວຍຂັ້ນຕອນການກະກຽມເພື່ອໃຫ້ແອັບອັດຕະໂນມັດແຈ້ງເຕືອນທ່ານກ່ອນວັນພິເສດເກີດຂຶ້ນ.

**ຂັ້ນຕອນຫຼັກ**:
1. ໄປທີ່ຟັງຊັນ → ວັນພິເສດ → ແຕະປຸ່ມ "+" (FAB)
2. ໃສ່ຊື່ວັນພິເສດ, ເລືອກວັນທີ (01/05), ເລືອກຊໍ້າຊ້ອນ "ລາຍປີ", ເລືອກເວລາແຈ້ງເຕືອນ (07:00)
3. ເພີ່ມຂັ້ນຕອນການກະກຽມ 1: "7 ວັນກ່ອນ – 08:00" - "ຊື້ຂອງຂວັນ"
4. ເພີ່ມຂັ້ນຕອນການກະກຽມ 2: "1 ວັນກ່ອນ – 19:00" - "ສັ່ງເຄັກ"
5. ແຕະ "ບັນທຶກ"

## 6. ເຫດຜົນ & ກົດລະບຽບ

### 6.1 ວັນທີແລະການຊໍ້າຊ້ອນ

- **ວັນທີ**: ເລືອກວັນ/ເດືອນ (ບໍ່ມີປີ), ສາມາດໃຊ້ປະຕິທິນຈັນທະຄະຕິ
- **ຊໍ້າຊ້ອນ**: ລາຍປີ (ຊໍ້າຊ້ອນທຸກປີ) ຫຼື ປີນີ້ເທົ່ານັ້ນ (ບໍ່ຊໍ້າຊ້ອນ)

### 6.2 ຂັ້ນຕອນການກະກຽມ

- ຂັ້ນຕອນສາມາດຖືກກໍາຫນົດເປັນ "X ວັນກ່ອນ" ຫຼື "ໃນມື້ນັ້ນ"
- ສາມາດເຊື່ອມກັບລາຍການກວດສອບການຊື້
- ສາມາດຕັ້ງຄ່າການແຈ້ງເຕືອນລາຍວັນຈົນກວ່າຈະສໍາເລັດ

### 6.3 ສະຖານະ

- **NOT_STARTED**: ບໍ່ເລີ່ມຕົ້ນ (ບໍ່ມີຂັ້ນຕອນທີ່ສໍາເລັດ)
- **IN_PROGRESS**: ກໍາລັງດໍາເນີນຢູ່ (ມີບາງຂັ້ນຕອນທີ່ສໍາເລັດ)
- **COMPLETED**: ສໍາເລັດ (ຂັ້ນຕອນທັງໝົດສໍາເລັດ)

### 6.4 ການແຈ້ງເຕືອນ

- ແອັບສົ່ງການແຈ້ງເຕືອນໃນວັນພິເສດຕາມເວລາທີ່ຕັ້ງໄວ້
- ສົ່ງການແຈ້ງເຕືອນສໍາລັບຂັ້ນຕອນການກະກຽມຕາມເວລາທີ່ຕັ້ງໄວ້

## 7. ໝາຍເຫດສໍາຄັນ

- **ປະຕິທິນຈັນທະຄະຕິ**: ສາມາດໃຊ້ປະຕິທິນຈັນທະຄະຕິສໍາລັບວັນພິເສດ, ແອັບຈະຄິດໄລ່ວັນທີສຸລິນທີ່ໃກ້ທີ່ສຸດອັດຕະໂນມັດ
- **ລາຍການກວດສອບ**: ສາມາດເຊື່ອມລາຍການກວດສອບການຊື້ກັບຂັ້ນຕອນການກະກຽມ
- **ການຊໍ້າຊ້ອນ**: ວັນພິເສດສາມາດຊໍ້າຊ້ອນລາຍປີຫຼືເປັນຄັ້ງດຽວ
