# 臨時収入

## 1. 目的

**臨時収入**モジュールは、以下のような固定サイクルがない非定期的な収入を記録するのに役立ちます：
- オンライン販売
- フリーランス
- ボーナス
- 現金ギフト
- その他の不規則な収入

**定期収入**とは異なり、臨時収入には自動サイクルがなく、毎回手動で入力する必要があります。

## 2. 使用するタイミング

以下の場合にこのモジュールを使用します：
- ランダムで非定期的な収入を記録
- 期間内の総収入を追跡
- 臨時収入の傾向を分析
- 月次予算に計算

## 3. 関連画面

- 臨時収入リスト
- 新しい収入の追加
- 収入の編集

## 4. 主な使用方法

### 4.1 臨時収入を追加

1. **機能** → **臨時収入**を選択
2. 右下の **+** (FAB) ボタンをタップ
3. 情報を入力：
   - **カテゴリ**: カテゴリを選択または新規作成
   - **金額**: 受領額を入力
   - **日付**: お金を受領した日を選択（デフォルトは今日）
   - **メモ**: 詳細な説明（任意）
4. **保存**をタップ

### 4.2 収入リストを表示

1. **機能** → **臨時収入**を選択
2. リストが設定されたレイアウト（1、2、3、または4列）に従って表示されます
3. **検索**を使用してカテゴリまたはメモでフィルタリング
4. **時間フィルタ**を選択: 今日 / 今週 / 今月 / 先月 / カスタム

### 4.3 収入を編集

1. 臨時収入リストに移動
2. 編集する項目を長押し
3. メニューから**編集**を選択
4. 情報を更新
5. **保存**をタップ

### 4.4 収入を削除

1. 臨時収入リストに移動
2. 削除する項目を長押し
3. メニューから**削除**を選択
4. 削除を確認

## 6. ロジックとルール

### 6.1 表示レイアウト

- 列数を設定できます: 1、2、3、または4列
- レイアウトは設定に保存され、すべての臨時収入リストに適用されます

### 6.2 時間フィルタ

- **今日**: 今日の収入のみ表示
- **今週**: 週の開始から今日まで
- **今月**: 月の開始から今日まで
- **先月**: 前月全体
- **カスタム**: カスタム時間範囲を選択

### 6.3 検索

- **カテゴリ名**と**メモ**で検索
- 大文字小文字を区別しない
- 入力に応じてリアルタイム検索

### 6.4 予算統合

- 臨時収入は予算の「臨時収入」に計算されます
- 月次総収入の追跡に役立ちます

## 7. 重要な注意事項

- **サイクルなし**: 臨時収入には自動サイクルがなく、毎回手動で入力する必要があります
- **削除可能**: 任意の収入を削除できます
- **予算統合**: 臨時収入は現在の月の予算に自動計算されます
