# 개인정보 보호정책

## 1. 소개

**Daily Life Assistant**는 사용자의 개인정보를 존중하는 것을 약속합니다. 본 개인정보 보호정책은 애플리케이션을 사용할 때 정보를 어떻게 처리하는지 설명합니다.

본 정책은 다음에 적용됩니다:

- Daily Life Assistant 애플리케이션
- 모든 플랫폼 (iOS, Android)

---

## 2. 수집되는 데이터

### 애플리케이션이 수집하지 않는 것

애플리케이션은 다음 정보를 **수집하지 않습니다**:

- 실명
- 이메일 (사용자가 적극적으로 지원에 연락하는 경우 제외)
- 전화번호
- 개인 식별 정보
- 실제 재정 데이터 (은행 정보, 신용카드)

> **원칙:** 애플리케이션은 **로컬 우선** 모델로 작동하며, 데이터는 기기에 저장되고 서버로 전송되지 않습니다.

---

## 3. 사용자가 입력하는 데이터

### 데이터 유형

애플리케이션을 사용할 때 다음과 같은 데이터를 입력할 수 있습니다:

- 지출 (수입, 일일 지출, 정기 지출)
- 작업 (할 일 목록, 특별한 날)
- 개인 메모
- 계획 (예산, 재정 목표)

### 데이터 저장

📌 **이 모든 데이터:**

- **기기에 저장**됩니다
- 애플리케이션 서버로 **전송되지 않습니다**
- 귀하만 액세스 권한을 가집니다

---

## 4. 백업 및 데이터 공유

### 백업

- 백업은 사용자가 **적극적으로 수행**합니다
- 백업 파일은 **사용자가 선택한 위치** (기기, iCloud, Google Drive 등의 개인 클라우드)에 저장됩니다
- 애플리케이션은 데이터를 자동으로 클라우드에 백업하지 않습니다

### 데이터 공유

- 데이터 공유는 사용자가 **적극적으로 내보내거나 공유**하는 경우에만 발생합니다
- 애플리케이션은 제3자와 데이터를 자동으로 공유하지 않습니다

---

## 5. 결제 데이터

### 프리미엄 결제

- 프리미엄 패키지 결제는 다음에 의해 처리됩니다:
  - **App Store** (iOS)
  - **Google Play** (Android)

### 결제 정보

애플리케이션은:

- 신용카드 정보를 **저장하지 않습니다**
- 결제 정보에 **액세스하지 않습니다**
- App Store/Google Play로부터 프리미엄 상태 확인만 받습니다

---

## 6. 분석 및 추적

### 광고

애플리케이션은:

- 광고를 **사용하지 않습니다**
- 광고 추적을 **사용하지 않습니다**
- 광고주와 데이터를 **공유하지 않습니다**

### 분석

- 애플리케이션은 익명 분석을 사용할 수 있습니다 (있는 경우)
- 분석에는 사용자의 **개인 데이터가 포함되지 않습니다**
- 현재 애플리케이션은 **분석을 사용하지 않습니다**

---

## 7. 사용자 권리

귀하는 다음 권리를 가집니다:

### 데이터 보기

- 애플리케이션에 입력한 모든 데이터 보기
- 데이터를 백업 파일로 내보내기

### 데이터 삭제

다음 방법으로 데이터를 삭제할 수 있습니다:

- 애플리케이션 내 각 항목을 수동으로 삭제
- 애플리케이션 제거 (기기의 모든 데이터가 삭제됩니다)

### 백업 제어

- 데이터 백업을 완전히 제어할 수 있습니다
- 백업 파일을 언제 어디에 저장할지 결정할 수 있습니다

---

## 8. 데이터 보안

### 보안 조치

- 데이터는 **운영 체제 암호화** (iOS/Android)로 보호됩니다
- 데이터는 기기의 **안전한 데이터베이스**에 저장됩니다
- 보안은 사용자의 **기기 보안** (비밀번호, Face ID, 지문)에 의존합니다

### 보안 제한

애플리케이션은 다음 경우 데이터를 보호할 수 없습니다:

- 기기가 손상되거나 분실된 경우
- 사용자가 백업 파일을 다른 사람과 공유한 경우
- 기기가 jailbreak (iOS) 또는 root (Android)된 경우

> **권장사항:** 강력한 비밀번호 및 기타 보안 조치로 기기를 항상 보호하세요.

---

## 9. 아동

- 애플리케이션은 13세 미만의 **아동을 대상으로 하지 않습니다**
- 애플리케이션은 아동으로부터 **의도적으로 데이터를 수집하지 않습니다**
- 부모이며 자녀가 개인정보를 제공한 것을 발견한 경우 문의하세요

---

## 10. 정책 변경

### 정책 업데이트

- 개인정보 보호정책은 시간이 지나면서 업데이트될 수 있습니다
- 정책의 새 버전은 **웹사이트** (GitHub Pages)에 게시됩니다
- 새 버전의 **시행일**이 명확히 명시됩니다

### 변경 알림

- 중요한 변경사항에 대해 알림을 시도합니다
- 정책이 업데이트된 후에도 애플리케이션을 계속 사용하는 것은 **새 정책에 동의한** 것으로 간주됩니다
