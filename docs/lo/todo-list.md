# ລາຍການວຽກ

## 1. ຈຸດປະສົງ

ໂມດູນ **ລາຍການວຽກ** ຊ່ວຍທ່ານຈັດການວຽກທີ່ຊໍ້າຊ້ອນແລະຕິດຕາມຄວາມຄືບໜ້າການສໍາເລັດ, ລວມທັງ:
- ຈັດການວຽກທີ່ຊໍ້າຊ້ອນ (ປະເພດ CYCLE ແລະ METRIC)
- ຕິດຕາມປະຫວັດການສໍາເລັດ
- ການແຈ້ງເຕືອນກ່ອນເວລາກໍານົດ
- ການຢືນຢັນການສໍາເລັດ (ສໍາລັບປະເພດ METRIC)

## 2. ເມື່ອໃດທີ່ຄວນໃຊ້

ໃຊ້ໂມດູນນີ້ເມື່ອທ່ານຕ້ອງການ:
- ຈັດການວຽກທີ່ຊໍ້າຊ້ອນ (ອາທິດ, ເດືອນ, ແລະອື່ນໆ)
- ຕິດຕາມວຽກທີ່ອີງຕາມຕົ ວຊີ້ວັດ (ຕົວຢ່າງ: ອອກກໍາລັງກາຍ 10 ຄັ້ງ)
- ໄດ້ຮັບການແຈ້ງເຕືອນກ່ອນເວລາກໍານົດ
- ຕິດຕາມປະຫວັດການສໍາເລັດ

## 3. ຫນ້າຈໍທີ່ກ່ຽວຂ້ອງ

- ລາຍການວຽກ
- ເພີ່ມວຽກໃໝ່
- ແກ້ໄຂວຽກ
- ຢືນຢັນການສໍາເລັດ (ສໍາລັບປະເພດ METRIC)
- ປະຫວັດວຽກ

## 4. ການນໍາໃຊ້ຫຼັກ

### 4.1 ເພີ່ມວຽກໃໝ່

1. ໄປທີ່ **ຟັງຊັນ** → ເລືອກ **ລາຍການວຽກ**
2. ແຕະປຸ່ມ **+** (FAB)
3. ເລືອກປະເພດ: **CYCLE** (ອີງຕາມເວລາ) ຫຼື **METRIC** (ອີງຕາມຕົວຊີ້ວັດ)
4. ຕື່ມຂໍ້ມູນ:
   - **ຊື່ວຽກ**: (ຈໍາເປັນ)
   - **ປະເພດ**: CYCLE ຫຼື METRIC
   - **ເວລາກໍານົດ**: (ສໍາລັບ CYCLE) ເລືອກວັນ/ເດືອນ
   - **ຈໍານວນເປົ້າຫມາຍ**: (ສໍາລັບ METRIC) ໃສ່ຈໍານວນ
   - **ເວລາແຈ້ງເຕືອນ**: ເລືອກເວລາ
   - **ໝາຍເຫດ**: (ທາງເລືອກ)
5. ແຕະ **ບັນທຶກ**

### 4.2 ຢືນຢັນການສໍາເລັດ (ສໍາລັບ METRIC)

1. ໄປທີ່ລາຍການວຽກ
2. ຊອກຫາວຽກປະເພດ METRIC ທີ່ກິດຈະກໍາ
3. ແຕະປຸ່ມ **ຢືນຢັນ** ໃນກາດ
4. ວຽກຖືກຢືນຢັນແລະປະຫວັດຖືກບັນທຶກ

### 4.3 ເບິ່ງປະຫວັດ

1. ໄປທີ່ລາຍການວຽກ
2. ແຕະທີ່ວຽກ
3. ເລືອກ **ປະຫວັດ** ເພື່ອເບິ່ງປະຫວັດການສໍາເລັດທັງໝົດ

### 4.4 ແກ້ໄຂວຽກ

1. ໄປທີ່ລາຍການວຽກ
2. ແຕະທີ່ວຽກເພື່ອແກ້ໄຂ
3. ແກ້ໄຂຂໍ້ມູນ
4. ແຕະ **ບັນທຶກ**

### 4.5 ປິດໃຊ້ງານ/ເປີດໃຊ້ງານວຽກ

1. ໄປທີ່ລາຍການວຽກ
2. ຊອກຫາວຽກທີ່ຕ້ອງການປິດໃຊ້ງານ/ເປີດໃຊ້ງານ
3. ສະຫຼັບ switch **ກິດຈະກໍາ** ຢູ່ດ້ານຂວາຂອງວຽກ

## 5. ຕົວຢ່າງ & ການສະແດງຜົນ UI

### TODO-01: ສ້າງວຽກ CYCLE (ອອກກໍາລັງກາຍລາຍອາທິດ)

**ເປົ້າຫມາຍ**: ສ້າງວຽກທີ່ຊໍ້າຊ້ອນຕາມເວລາເພື່ອຕິດຕາມການອອກກໍາລັງກາຍລາຍອາທິດ.

**ຂັ້ນຕອນ**:
1. ໄປທີ່ຟັງຊັນ → ລາຍການວຽກ → ແຕະ "+"
2. ເລືອກປະເພດ "CYCLE"
3. ໃສ່ຊື່: "ອອກກໍາລັງກາຍ"
4. ເລືອກເວລາກໍານົດ: ທຸກໆວັນອາທິດ
5. ເລືອກເວລາແຈ້ງເຕືອນ: 06:00
6. ແຕະ "ບັນທຶກ"

## 6. ເຫດຜົນ & ກົດລະບຽບ

### 6.1 ປະເພດວຽກ

- **CYCLE**: ວຽກທີ່ຊໍ້າຊ້ອນຕາມເວລາ (ອາທິດ, ເດືອນ, ແລະອື່ນໆ)
- **METRIC**: ວຽກທີ່ອີງຕາມຕົວຊີ້ວັດ (ຕົວຢ່າງ: ອອກກໍາລັງກາຍ 10 ຄັ້ງ)

### 6.2 ສະຖານະ

- **PENDING**: ລໍຖ້າ (ຍັງບໍ່ໄດ້ສໍາເລັດ)
- **OVERDUE**: ເກີນກໍານົດ (ບໍ່ໄດ້ສໍາເລັດໃນເວລາກໍານົດ)
- **NOT_COMPLETED**: ບໍ່ສໍາເລັດ (ສໍາລັບ CYCLE ທີ່ບໍ່ໄດ້ສໍາເລັດ)
- **COMPLETED**: ສໍາເລັດ
- **CANCELLED**: ຍົກເລີກ
- **INACTIVE**: ບໍ່ກິດຈະກໍາ

### 6.3 ການຢືນຢັນ

- ສໍາລັບປະເພດ METRIC, ທ່ານສາມາດຢືນຢັນການສໍາເລັດໂດຍການແຕະປຸ່ມ "ຢືນຢັນ"
- ແຕ່ລະການຢືນຢັນຖືກບັນທຶກໃນປະຫວັດ

### 6.4 ການແຈ້ງເຕືອນ

- ແອັບສົ່ງການແຈ້ງເຕືອນກ່ອນເວລາກໍານົດ
- ເວລາແຈ້ງເຕືອນສາມາດຕັ້ງຄ່າໄດ້ສໍາລັບແຕ່ລະວຽກ

## 7. ໝາຍເຫດສໍາຄັນ

- **ປະເພດ CYCLE**: ວຽກທີ່ຊໍ້າຊ້ອນຕາມເວລາ, ສາມາດຕິດຕາມຕາມວັນທີ
- **ປະເພດ METRIC**: ວຽກທີ່ອີງຕາມຕົວຊີ້ວັດ, ຕ້ອງຢືນຢັນການສໍາເລັດແຕ່ລະຄັ້ງ
- **ການຢືນຢັນ**: ສໍາລັບປະເພດ METRIC, ທ່ານສາມາດຢືນຢັນການສໍາເລັດໂດຍການແຕະປຸ່ມ "ຢືນຢັນ" ໃນກາດ
