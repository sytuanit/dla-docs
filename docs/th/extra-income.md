# รายได้พิเศษ

## 1. วัตถุประสงค์

โมดูล **รายได้พิเศษ** ช่วยให้คุณบันทึกรายได้ที่ไม่ประจำที่ไม่มีรอบคงที่ เช่น:
- ขายออนไลน์
- ฟรีแลนซ์
- โบนัส
- เงินของขวัญ
- รายได้ไม่ปกติอื่นๆ

ไม่เหมือนกับ **รายได้ประจำ** รายได้พิเศษไม่มีรอบอัตโนมัติ คุณต้องกรอกด้วยตนเองทุกครั้ง

## 2. เมื่อใดควรใช้

ใช้โมดูลนี้เมื่อคุณต้องการ:
- บันทึกรายได้แบบสุ่มที่ไม่ประจำ
- ติดตามรายได้รวมในช่วงเวลาหนึ่ง
- วิเคราะห์แนวโน้มรายได้พิเศษ
- คำนวณเข้างบประมาณรายเดือน

## 3. หน้าจอที่เกี่ยวข้อง

- รายการรายได้พิเศษ
- เพิ่มรายได้ใหม่
- แก้ไขรายได้

## 4. การใช้งานหลัก

### 4.1 เพิ่มรายได้พิเศษ

1. ไปที่ **ฟังก์ชัน** → เลือก **รายได้พิเศษ**
2. แตะปุ่ม **+** (FAB) ที่มุมล่างขวา
3. กรอกข้อมูล:
   - **หมวดหมู่**: เลือกหรือสร้างหมวดหมู่ใหม่
   - **จำนวนเงิน**: ใส่จำนวนที่ได้รับ
   - **วันที่**: เลือกวันที่ได้รับเงิน (ค่าเริ่มต้นคือวันนี้)
   - **หมายเหตุ**: คำอธิบายรายละเอียด (ไม่บังคับ)
4. แตะ **บันทึก**

### 4.2 ดูรายการรายได้

1. ไปที่ **ฟังก์ชัน** → เลือก **รายได้พิเศษ**
2. รายการแสดงตามเลย์เอาต์ที่คุณกำหนด (1, 2, 3, หรือ 4 คอลัมน์)
3. ใช้ **ค้นหา** เพื่อกรองตามหมวดหมู่หรือหมายเหตุ
4. เลือก **ตัวกรองเวลา**: วันนี้ / สัปดาห์นี้ / เดือนนี้ / เดือนที่แล้ว / กำหนดเอง

### 4.3 แก้ไขรายได้

1. ไปที่รายการรายได้พิเศษ
2. กดค้างที่รายการเพื่อแก้ไข
3. เลือก **แก้ไข** จากเมนู
4. อัปเดตข้อมูล
5. แตะ **บันทึก**

### 4.4 ลบรายได้

1. ไปที่รายการรายได้พิเศษ
2. กดค้างที่รายการเพื่อลบ
3. เลือก **ลบ** จากเมนู
4. ยืนยันการลบ

## 6. ตรรกะและกฎ

### 6.1 เลย์เอาต์การแสดงผล

- คุณสามารถกำหนดจำนวนคอลัมน์: 1, 2, 3, หรือ 4 คอลัมน์
- เลย์เอาต์จะถูกบันทึกในการตั้งค่าและใช้กับรายการรายได้พิเศษทั้งหมด

### 6.2 ตัวกรองเวลา

- **วันนี้**: แสดงเฉพาะรายได้จากวันนี้
- **สัปดาห์นี้**: จากต้นสัปดาห์ถึงวันนี้
- **เดือนนี้**: จากต้นเดือนถึงวันนี้
- **เดือนที่แล้ว**: ทั้งเดือนก่อนหน้า
- **กำหนดเอง**: เลือกช่วงเวลาที่กำหนดเอง

### 6.3 ค้นหา

- ค้นหาใน **ชื่อหมวดหมู่** และ **หมายเหตุ**
- ไม่สนใจตัวพิมพ์เล็ก-ใหญ่
- ค้นหาแบบเรียลไทม์ขณะพิมพ์

### 6.4 การรวมเข้างบประมาณ

- รายได้พิเศษจะถูกคำนวณเข้า "รายได้พิเศษ" ในงบประมาณ
- ช่วยให้คุณติดตามรายได้รวมรายเดือน

## 7. หมายเหตุสำคัญ

- **ไม่มีรอบ**: รายได้พิเศษไม่มีรอบอัตโนมัติ คุณต้องกรอกด้วยตนเองทุกครั้ง
- **สามารถลบได้**: คุณสามารถลบรายได้ใดๆ ได้
- **การรวมเข้างบประมาณ**: รายได้พิเศษจะถูกคำนวณเข้างบประมาณเดือนปัจจุบันอัตโนมัติ
