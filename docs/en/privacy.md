# Privacy Policy

## 1. Introduction

**Daily Life Assistant** is committed to respecting user privacy. This privacy policy explains how we handle your information when using the application.

This policy applies to:

- Daily Life Assistant application
- All platforms (iOS, Android)

---

## 2. Data Collected

### Application DOES NOT Collect

The application **does not collect** the following information:

- Real name
- Email (except when user actively contacts support)
- Phone number
- Personal identifiers
- Actual financial data (bank information, credit cards)

> **Principle:** The application operates on a **local-first** model, meaning data is stored on your device, not sent to a server.

---

## 3. User-Entered Data

### Data Types

When using the application, you may enter data such as:

- Expenses (income, daily expenses, recurring expenses)
- Tasks (todos, special occasions)
- Personal notes
- Plans (budgets, financial goals)

### Data Storage

- Is stored **on your device**

---

## 4. Backup & Data Sharing

### Backup

- Backup is **actively performed** by the user
- Backup files are saved where **the user chooses** (device, personal cloud like iCloud, Google Drive)
- The application does not automatically backup data to cloud

### Data Sharing

- Data sharing only occurs when the user **actively exports or shares** backup files
- The application does not automatically share data with third parties

---

## 5. Payment Data

### Premium Payment

- Payment for Premium package is processed by:
  - **App Store** (iOS)
  - **Google Play** (Android)

### Payment Information

The application:

- **Does not store** credit card information
- **Does not access** your payment information
- Only receives confirmation about Premium status from App Store/Google Play

---

## 6. Analytics & Tracking

### Advertising

The application:

- **Does not use** advertising
- **Does not use** advertising tracking
- **Does not share** data with advertisers

### Analytics

- The application may use anonymous analytics (if any)
- Analytics **does not contain personal data** of users
- Currently, the application **does not use analytics**

---

## 7. User Rights

You have the right to:

### View Data

- View all data you have entered in the application
- Export data as backup files

### Delete Data

You can delete data by:

- Manually deleting each item in the application
- Uninstalling the application (will delete all data on the device)

### Control Backup

- You have complete control over data backup
- You decide when and where to save backup files

---

## 8. Data Security

### Security Measures

- Data is protected by **operating system encryption** (iOS/Android)
- Data is stored in a **secure database** on your device
- Security depends on **device security** of the user (password, Face ID, fingerprint)

### Security Limitations

The application cannot protect data if:

- Device is compromised or lost
- User shares backup files with others
- Device is jailbroken (iOS) or rooted (Android)

> **Recommendation:** Always protect your device with a strong password and other security measures.

---

## 9. Children

- The application **is not directed at children** under 13 years old
- The application **does not intentionally collect** data from children
- If you are a parent and discover your child has provided personal information, please contact us

---

## 10. Policy Changes

### Policy Updates

- Privacy policy may be updated over time
- New versions of the policy will be published on the **website** (GitHub Pages)
- **Effective date** of the new version will be clearly stated

### Change Notification

- We will try to notify about important changes
- Continued use of the application after the policy is updated is understood as **agreement to the new policy**

## 11. Contact

### How to Contact

If you have questions about this privacy policy, please contact via:

- **Support email** (see in the Settings section of the application)

### Information Security When Contacting

- The application **does not require** you to provide sensitive data when contacting support
- Only provide information necessary to resolve your issue

## Summary

This privacy policy aims to:

- **Transparency** about how the application handles your data
- **Protect user privacy** rights
- **Comply** with Apple App Privacy and Google Data Safety requirements

**Core Principle:** Your data belongs to you. The application is just a tool for you to manage data on your device.
