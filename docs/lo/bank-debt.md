# ເງິນກູ້ທະນາຄານ

## 1. ຈຸດປະສົງ

ໂມດູນ **ເງິນກູ້ທະນາຄານ** ຊ່ວຍທ່ານຈັດການເງິນກູ້ທະນາຄານ, ລວມທັງ:
- ຕິດຕາມຈໍານວນເງິນກູ້, ອັດຕາດອກເບ້ຍ, ເວລາກໍານົດ
- ຈັດການຕາຕະລາງການຊໍາລະ
- ຄິດໄລ່ດອກເບ້ຍຕາມຊ່ວງເວລາ (ຖ້າມີ)
- ຈັດການຄ່າປັບໃໝ່ການຊໍາລະຊ້າ
- ການຊໍາລະກ່ອນເວລາ (ຖ້າຈໍາເປັນ)

## 2. ເມື່ອໃດທີ່ຄວນໃຊ້

ໃຊ້ໂມດູນນີ້ເມື່ອທ່ານມີ:
- ເງິນກູ້ທະນາຄານ
- ຕ້ອງການຕິດຕາມຕາຕະລາງການຊໍາລະ
- ຕ້ອງການຄິດໄລ່ດອກເບ້ຍແລະຄ່າປັບໃໝ່
- ຕ້ອງການການແຈ້ງເຕືອນເມື່ອເງິນກູ້ຄວນຊໍາລະ

## 3. ຫນ້າຈໍທີ່ກ່ຽວຂ້ອງ

- ລາຍການເງິນກູ້
- ເພີ່ມເງິນກູ້ໃໝ່ (4 ຂັ້ນຕອນ)
- ແກ້ໄຂເງິນກູ້
- ລາຍລະອຽດເງິນກູ້ແລະຕາຕະລາງການຊໍາລະ
- ການຊໍາລະກ່ອນເວລາ

## 4. ການນໍາໃຊ້ຫຼັກ

### 4.1 ເພີ່ມເງິນກູ້ໃໝ່ (4 ຂັ້ນຕອນ)

#### ຂັ້ນຕອນ 1: ຂໍ້ມູນພື້ນຖານ

1. ໄປທີ່ **ຟັງຊັນ** → ເລືອກ **ເງິນກູ້ທະນາຄານ**
2. ແຕະປຸ່ມ **+** (FAB)
3. ຕື່ມຂໍ້ມູນ:
   - **ທະນາຄານ**: ເລືອກຫຼືສ້າງທະນາຄານໃໝ່
   - **ຊື່ເງິນກູ້**: (ຕົວຢ່າງ: "ເງິນກູ້ບ້ານ")
   - **ຈໍານວນເງິນກູ້**: ຈໍານວນຕົ້ນທຶນ
   - **ວັນທີຈໍານວນເງິນ**: ວັນທີໄດ້ຮັບເງິນ
   - **ເວລາກໍານົດ**: ຈໍານວນປີ
   - **ປະເພດດອກເບ້ຍ**: ອັດຕາສໍາເນົາ/ລອຍຫຼືອັດຕາຄົງທີ່
4. ແຕະ **ຕໍ່ໄປ**

#### ຂັ້ນຕອນ 2: ຕັ້ງຄ່າອັດຕາດອກເບ້ຍ

**ຖ້າເລືອກ "ອັດຕາສໍາເນົາ/ລອຍ":**
- ເປີດໃຊ້ **ມີອັດຕາສໍາເນົາ** (ຖ້າມີ)
- ໃສ່ **ເດືອນສໍາເນົາ** ແລະ **ອັດຕາສໍາເນົາ**
- ເພີ່ມຊ່ວງເວລາອັດຕາລອຍ:
  - ເລືອກປີແລະຊ່ວງເດືອນ
  - ໃສ່ອັດຕາດອກເບ້ຍ (%/ປີ)
  - ເລືອກ **ລອຍ** ຫຼື **ຄົງທີ່**

**ຖ້າເລືອກ "ອັດຕາຄົງທີ່":**
- ໃສ່ **ອັດຕາຄົງທີ່** (%/ປີ)

ແຕະ **ຕໍ່ໄປ**

#### ຂັ້ນຕອນ 3: ຕັ້ງຄ່າຄ່າປັບໃໝ່

1. ເປີດໃຊ້ **ມີຄ່າປັບໃໝ່ການຊໍາລະຊ້າ** (ຖ້າມີ)
2. ເພີ່ມຊ່ວງເວລາຄ່າປັບໃໝ່:
   - ເລືອກປີແລະຊ່ວງເດືອນ
   - ໃສ່ **ອັດຕາຄ່າປັບໃໝ່** (%/ປີ)
3. ແຕະ **ຕໍ່ໄປ**

#### ຂັ້ນຕອນ 4: ຢືນຢັນແລະບັນທຶກ

1. ກວດສອບຂໍ້ມູນ:
   - ຈໍານວນລວມທີ່ຕ້ອງຊໍາລະ
   - ຕາຕະລາງການຊໍາລະທີ່ຄາດຫວັງ
2. ແຕະ **ບັນທຶກ**

### 4.2 ເບິ່ງລາຍລະອຽດເງິນກູ້

1. ໄປທີ່ລາຍການເງິນກູ້
2. ແຕະທີ່ເງິນກູ້
3. ເບິ່ງຂໍ້ມູນ:
   - ຂໍ້ມູນພື້ນຖານ
   - ຕາຕະລາງການຊໍາລະ
   - ຈໍານວນຊໍາລະແລ້ວ / ເຫຼືອ
   - ອັດຕາດອກເບ້ຍແລະຄ່າປັບໃໝ່

### 4.3 ກໍາຫນົດຊ່ວງເວລາການຊໍາລະເປັນຊໍາລະແລ້ວ

1. ໄປທີ່ລາຍລະອຽດເງິນກູ້
2. ຊອກຫາຊ່ວງເວລາການຊໍາລະທີ່ຄວນຊໍາລະ (badge "ບໍ່ໄດ້ຊໍາລະ")
3. ແຕະ **ກໍາຫນົດເປັນຊໍາລະແລ້ວ**
4. ຕື່ມຂໍ້ມູນ:
   - **ວັນທີຊໍາລະຕົວຈິງ**: ວັນທີຊໍາລະ (ເລີ່ມຕົ້ນ = ມື້ນີ້)
   - **ດອກເບ້ຍຕົວຈິງທີ່ຊໍາລະ**: ດອກເບ້ຍຕົວຈິງທີ່ຊໍາລະ (ເລີ່ມຕົ້ນ = ດອກເບ້ຍທີ່ວາງແຜນ)
   - **ໝາຍເຫດ**: (ທາງເລືອກ)
5. ເບິ່ງ **ລວມການຊໍາລະຕົວຈິງ** ຄິດໄລ່ອັດຕະໂນມັດ (ຕົ້ນທຶນ + ດອກເບ້ຍຕົວຈິງ)
6. ແຕະ **ຢືນຢັນ**

### 4.4 ອັບເດດອັດຕາດອກເບ້ຍປັດຈຸບັນ

1. ໄປທີ່ລາຍລະອຽດເງິນກູ້ (ສະແດງພຽງແຕ່ຖ້າປັດຈຸບັນຢູ່ໃນຊ່ວງເວລາອັດຕາລອຍ)
2. ແຕະ **ອັບເດດອັດຕາດອກເບ້ຍປັດຈຸບັນ**
3. ຕື່ມຂໍ້ມູນ:
   - **ອັດຕາດອກເບ້ຍໃໝ່**: ອັດຕາດອກເບ້ຍໃໝ່ (%/ປີ)
   - **ວັນທີມີຜົນ**: ວັນທີເລີ່ມຕົ້ນນໍາໃຊ້ອັດຕາໃໝ່ (ເລີ່ມຕົ້ນ = ຕົ້ນຊ່ວງເວລາປັດຈຸບັນ)
   - **ໝາຍເຫດ**: (ທາງເລືອກ)
4. ແຕະ **ບັນທຶກ**
5. ຊ່ວງເວລາທີ່ບໍ່ໄດ້ຊໍາລະຈາກຊ່ວງເວລາປັດຈຸບັນເປັນຕົ້ນໄປຈະຖືກອັບເດດດ້ວຍອັດຕາດອກເບ້ຍໃໝ່

### 4.5 ການຊໍາລະກ່ອນເວລາ

1. ໄປທີ່ລາຍລະອຽດເງິນກູ້
2. ແຕະ **ຄິດໄລ່ຈໍານວນການຊໍາລະ**
3. **ຂັ້ນຕອນ 1 - ໃສ່ຂໍ້ມູນການຊໍາລະລ່ວງໜ້າ:**
   - ເລືອກວິທີ: **ການຊໍາລະບາງສ່ວນ** ຫຼື **ການຊໍາລະທັງໝົດ**
   - ເລືອກວັນທີຊໍາລະລ່ວງໜ້າ (ເລີ່ມຕົ້ນ = ມື້ນີ້)
   - ໃສ່ຈໍານວນຊໍາລະລ່ວງໜ້າ (ຖ້າບາງສ່ວນ)
   - ເບິ່ງ **ຄ່າປັບໃໝ່ການຊໍາລະລ່ວງໜ້າ** ຄິດໄລ່ອັດຕະໂນມັດ
4. ແຕະ **ຕໍ່ໄປ**
5. **ຂັ້ນຕອນ 2 - ປຽບທຽບທາງເລືອກ:**
   - ເບິ່ງການປຽບທຽບລະຫວ່າງ "ບໍ່ມີການຊໍາລະລ່ວງໜ້າ" ແລະ "ການຊໍາລະລ່ວງໜ້າ"
   - ເບິ່ງຜົນລັບ: ການປະຫຍັດດອກເບ້ຍ, ການຫຼຸດເວລາ
6. ແຕະ **ຢືນຢັນການຊໍາລະລ່ວງໜ້າ**

### 4.6 ແກ້ໄຂເງິນກູ້

1. ໄປທີ່ລາຍລະອຽດເງິນກູ້
2. ແຕະ **ແກ້ໄຂ** (ແກ້ໄຂພຽງແຕ່ຊື່, ໝາຍເຫດ, ທະນາຄານ)
3. ແກ້ໄຂຂໍ້ມູນທີ່ສາມາດແກ້ໄຂ:
   - **ຊື່ເງິນກູ້**: ສາມາດແກ້ໄຂ
   - **ທະນາຄານ**: ສາມາດປ່ຽນ
   - **ໝາຍເຫດ**: ສາມາດແກ້ໄຂ
   - **ຈໍານວນເງິນກູ້, ວັນທີຈໍານວນເງິນ, ເວລາກໍານົດ, ອັດຕາດອກເບ້ຍ**: ສາມາດແກ້ໄຂໄດ້ພຽງແຕ່ຖ້າຍັງບໍ່ໄດ້ຊໍາລະ
4. ແຕະ **ບັນທຶກ**

## 5. ຕົວຢ່າງ & ການສະແດງຜົນ UI

### LOAN-01: ສ້າງເງິນກູ້ໃໝ່ (ເງິນກູ້ບ້ານດ້ວຍອັດຕາດອກເບ້ຍສໍາເນົາ)

**ເປົ້າຫມາຍ**: ສ້າງເງິນກູ້ໃໝ່ເພື່ອຕິດຕາມເງິນກູ້ບ້ານ, ອັດຕາດອກເບ້ຍສໍາເນົາ, ແລະຕາຕະລາງການຊໍາລະລາຍເດືອນ.

**ຂັ້ນຕອນ**:
1. ໄປທີ່ **ຟັງຊັນ** → ເລືອກ **ເງິນກູ້ທະນາຄານ**
2. ແຕະປຸ່ມ **+** (FAB) ເພື່ອເພີ່ມເງິນກູ້ໃໝ່
3. **ຂັ້ນຕອນ 1 - ຂໍ້ມູນພື້ນຖານ:**
   - ເລືອກທະນາຄານ: ທະນາຄານອາເມລິກາ
   - ໃສ່ຊື່: "ເງິນກູ້ບ້ານ - ອາພາດເມັນຕົວເມືອງ"
   - ໃສ່ຈໍານວນເງິນກູ້: 4,000,000,000 ກີບ
   - ເລືອກວັນທີຈໍານວນເງິນ: 04/01/2023
   - ໃສ່ເວລາກໍານົດ: 10 ປີ (ຄິດໄລ່ອັດຕະໂນມັດ = 120 ຊ່ວງເວລາ)
   - ເລືອກເວລາແຈ້ງເຕືອນ: 10:00 ແລະ 19:00
   - ເລືອກປະເພດດອກເບ້ຍ: "ຍອດເງິນຫຼຸດລົງ"
   - ແຕະ **ຕໍ່ໄປ**
4. **ຂັ້ນຕອນ 2 - ຕັ້ງຄ່າອັດຕາດອກເບ້ຍ:**
   - ເປີດໃຊ້ "ມີຊ່ວງເວລາດອກເບ້ຍສໍາເນົາ"
   - ໃສ່: 6 ເດືອນທໍາອິດ @ 6.0%/ປີ
   - ເພີ່ມຊ່ວງເວລາຕໍ່ມາ:
     - ປີທີ 1 (ເດືອນ 7-12): 9.0%/ປີ, ລອຍ
     - ປີທີ 2 (ເດືອນ 13-24): 9.5%/ປີ, ລອຍ
     - ປີທີ 3 ເປັນຕົ້ນໄປ: 10.0%/ປີ, ລອຍ
   - ແຕະ **ຕໍ່ໄປ**
5. **ຂັ້ນຕອນ 3 - ຕັ້ງຄ່າຄ່າປັບໃໝ່ການຊໍາລະລ່ວງໜ້າ:**

   - ໃສ່ຄ່າປັບໃໝ່: ປີທີ 1-3: 2.0%, ປີທີ 4-5: 1.5%, ປີທີ 6+: 1.0%
   - ແຕະ **ຕໍ່ໄປ**
6. **ຂັ້ນຕອນ 4 - ຢືນຢັນ:**

   - ແຕະ **ສ້າງເງິນກູ້**

**ຜົນລັບ**: ເງິນກູ້ຖືກສ້າງສໍາເລັດ, ຕາຕະລາງການຊໍາລະ 120 ຊ່ວງເວລາຖືກສ້າງອັດຕະໂນມັດ, ການແຈ້ງເຕືອນຖືກກໍາຫນົດ.

### LOAN-02: ເບິ່ງລາຍການເງິນກູ້ແລະລາຍລະອຽດ

**ເປົ້າຫມາຍ**: ເບິ່ງສະຫຼຸບເງິນກູ້, ກອງຕາມສະຖານະ, ຄົ້ນຫາ, ແລະເບິ່ງລາຍລະອຽດຂອງແຕ່ລະເງິນກູ້.

**ຂັ້ນຕອນ**:

2. ເບິ່ງຫນ້າຈໍລາຍການດ້ວຍຕົວກອງ "ກິດຈະກໍາ" (ເລີ່ມຕົ້ນ) ແລະ "ສໍາເລັດ"
3. ສະຫຼັບລະຫວ່າງຕົວກອງເພື່ອເບິ່ງສະຫຼຸບທີ່ແຕກຕ່າງກັນ
4. ໃຊ້ແຖບຄົ້ນຫາ: ໃສ່ "ຕົວເມືອງ"

7. ໃຊ້ແຖບຄົ້ນຫາໃນຕາຕະລາງການຊໍາລະ: ໃສ່ "9/2024"

**ຜົນລັບ**: ລາຍການສະແດງຢ່າງຖືກຕ້ອງຕາມຕົວກອງ, ລາຍລະອຽດເງິນກູ້ສະແດງຂໍ້ມູນເຕັມແລະຕາຕະລາງການຊໍາລະ.

## 6. ເຫດຜົນ & ກົດລະບຽບ

### 6.1 ອັດຕາສໍາເນົາ/ລອຍ

- ສາມາດມີຊ່ວງເວລາສໍາເນົາ (ອັດຕາດອກເບ້ຍຕ່ໍາກວ່າ)
- ຫຼັງຈາກຊ່ວງເວລາສໍາເນົາ, ອັດຕາດອກເບ້ຍລອຍຕາມຊ່ວງເວລາ
- ແຕ່ລະຊ່ວງເວລາສາມາດເປັນ **ລອຍ** (ອີງຕາມຕະຫຼາດ) ຫຼື **ຄົງທີ່**

### 6.2 ຄ່າປັບໃໝ່ການຊໍາລະຊ້າ

- ຄ່າປັບໃໝ່ຄິດໄລ່ຕາມ %/ປີ
- ສາມາດຕັ້ງຄ່າແຕກຕ່າງກັນສໍາລັບແຕ່ລະຊ່ວງເວລາ
- ຄ່າປັບໃໝ່ນໍາໃຊ້ພຽງແຕ່ເມື່ອການຊໍາລະຊ້າ

### 6.3 ຕາຕະລາງການຊໍາລະ

- ແອັບສ້າງຕາຕະລາງການຊໍາລະອັດຕະໂນມັດຕາມ:
  - ຈໍານວນເງິນກູ້
  - ອັດຕາດອກເບ້ຍ
  - ເວລາກໍານົດ
- ແຕ່ລະຊ່ວງເວລາການຊໍາລະລວມທັງ: ຕົ້ນທຶນ + ດອກເບ້ຍ

### 6.4 ການຊໍາລະກ່ອນເວລາ

- ຄິດໄລ່ຈໍານວນເຫຼືອ (ຕົ້ນທຶນ + ດອກເບ້ຍ + ຄ່າປັບໃໝ່ຖ້າມີ)
- ຫຼັງຈາກການຊໍາລະ, ເງິນກູ້ຈະປ່ຽນເປັນສະຖານະ "ສໍາເລັດ"

### 6.5 ການແຈ້ງເຕືອນ

- ແອັບສົ່ງການແຈ້ງເຕືອນເມື່ອເງິນກູ້ຄວນຊໍາລະ
- ເວລາແຈ້ງເຕືອນສາມາດຕັ້ງຄ່າໄດ້ສໍາລັບແຕ່ລະເງິນກູ້ (`notificationTime1`, `notificationTime2`, ເລີ່ມຕົ້ນ 10:00 ແລະ 19:00)

## 7. ໝາຍເຫດສໍາຄັນ

- **ອັດຕາດອກເບ້ຍສະລັບສັບຊ້ອນ**: ໂມດູນນີ້ສະໜັບສະໜູນອັດຕາດອກເບ້ຍທີ່ປ່ຽນຕາມຊ່ວງເວລາ, ຕ້ອງການການຕັ້ງຄ່າຢ່າງລະມັດລະວັງ
- **ບໍ່ສາມາດລຶບເມື່ອມີຕາຕະລາງການຊໍາລະ**: ຖ້າມີຕາຕະລາງການຊໍາລະ, ທ່ານສາມາດຊໍາລະເທົ່ານັ້ນ, ບໍ່ສາມາດລຶບ
- **ການຊໍາລະກ່ອນເວລາ**: ອາດຈະຕ້ອງການຄ່າປັບໃໝ່ເພີ່ມເຕີມ, ຂຶ້ນກັບນະໂຍບາຍທະນາຄານ
- **ຕາຕະລາງການຊໍາລະ**: ຕາຕະລາງການຊໍາລະຖືກຄິດໄລ່ອັດຕະໂນມັດ, ທ່ານບໍ່ສາມາດແກ້ໄຂໂດຍກົງ
