# ລາຍໄດ້ປົກກະຕິ

## 1. ຈຸດປະສົງ

ໂມດູນ **ລາຍໄດ້ປົກກະຕິ** ຊ່ວຍທ່ານຈັດການແຫຼ່ງລາຍໄດ້ປົກກະຕິເຊັ່ນ:
- ເງິນເດືອນ
- ເງິນເຊົ່າ
- ເງິນບໍານານ
- ດອກເບ້ຍການລົງທຶນ
- ລາຍໄດ້ປົກກະຕິອື່ນໆ

ໂມດູນນີ້ສ້າງ **ການເກີດຂຶ້ນ** ອັດຕະໂນມັດຕາມວົງຈອນທີ່ທ່ານຕັ້ງຄ່າ, ແລະແຈ້ງເຕືອນທ່ານເມື່ອເຖິງເວລາຮັບເງິນ.

## 2. ເມື່ອໃດທີ່ຄວນໃຊ້

ໃຊ້ໂມດູນນີ້ເມື່ອທ່ານມີ:
- ລາຍໄດ້ຄົງທີ່ຕາມຕາຕະລາງ (ອາທິດ, ສອງອາທິດ, ຫຼືລາຍເດືອນ)
- ຕ້ອງການຕິດຕາມແລະຢືນຢັນເມື່ອໄດ້ຮັບເງິນ
- ຕ້ອງການການຄິດໄລ່ອັດຕະໂນມັດເຂົ້າໃນງົບປະມານລາຍເດືອນ

## 3. ຫນ້າຈໍທີ່ກ່ຽວຂ້ອງ

- ລາຍການລາຍໄດ້ປົກກະຕິ
- ເພີ່ມລາຍໄດ້ປົກກະຕິໃໝ່
- ແກ້ໄຂລາຍໄດ້ປົກກະຕິ
- ປະຫວັດການເກີດຂຶ້ນ

## 4. ການນໍາໃຊ້ຫຼັກ

### 4.1 ເພີ່ມລາຍໄດ້ປົກກະຕິໃໝ່

1. ໄປທີ່ **ຟັງຊັນ** → ເລືອກ **ລາຍໄດ້ປົກກະຕິ**
2. ແຕະປຸ່ມ **+** (FAB) ຢູ່ມຸມລຸ່ມຂວາ
3. ຕື່ມຂໍ້ມູນ:
   - **ປະເພດ**: ເລືອກຫຼືສ້າງປະເພດໃໝ່
   - **ຈໍານວນ**: ໃສ່ຈໍານວນລາຍໄດ້ (ສາມາດປ່ອຍໄວ້ວ່າງ, ໃສ່ເມື່ອຢືນຢັນ)
   - **ວົງຈອນ**: ເລືອກ ອາທິດ / ສອງອາທິດ / ລາຍເດືອນ
   - **ວັນທີ**: ເລືອກວັນທີໃນວົງຈອນ (ຕົວຢ່າງ: ວັນທີ 5 ຂອງແຕ່ລະເດືອນ)
   - **ວັນທີເລີ່ມຕົ້ນ**: (ສໍາລັບວົງຈອນສອງອາທິດເທົ່ານັ້ນ) ເລືອກວັນທີເລີ່ມຕົ້ນ
   - **ໝາຍເຫດ**: ຂໍ້ມູນເພີ່ມເຕີມ (ທາງເລືອກ)
4. ແຕະ **ບັນທຶກ**

### 4.2 ຢືນຢັນການໄດ້ຮັບເງິນ

1. ໄປທີ່ລາຍການລາຍໄດ້ປົກກະຕິ
2. ຊອກຫາລາຍການທີ່ມີ badge **"ລໍຖ້າການຢືນຢັນ"** (ສີເຫຼືອງ)
3. ແຕະທີ່ລາຍການເພື່ອເປີດການສົນທະນາຢືນຢັນ
4. ຕື່ມ:
   - **ຈໍານວນຕົວຈິງ**: (ຖ້າຕ່າງຈາກທີ່ຄາດຫວັງ)
   - **ໝາຍເຫດ**: (ທາງເລືອກ)
5. ແຕະ **ຢືນຢັນ**

### 4.3 ແກ້ໄຂລາຍໄດ້ປົກກະຕິ

1. ໄປທີ່ລາຍການລາຍໄດ້ປົກກະຕິ
2. ແຕະທີ່ລາຍການທີ່ຕ້ອງການແກ້ໄຂ
3. ເລືອກ **ແກ້ໄຂ** ຈາກເມນູ
4. ອັບເດດຂໍ້ມູນ
5. ແຕະ **ບັນທຶກ**

### 4.4 ເບິ່ງປະຫວັດ

1. ໄປທີ່ລາຍການລາຍໄດ້ປົກກະຕິ
2. ແຕະທີ່ລາຍການ
3. ເລືອກ **ປະຫວັດ** ເພື່ອເບິ່ງການເກີດຂຶ້ນທັງໝົດໃນອະດີດ

### 4.5 ປິດໃຊ້ງານ/ເປີດໃຊ້ງານລາຍໄດ້

1. ໄປທີ່ລາຍການລາຍໄດ້ປົກກະຕິ
2. ຊອກຫາລາຍການທີ່ຕ້ອງການປິດໃຊ້ງານ/ເປີດໃຊ້ງານ
3. ສະຫຼັບ **ກິດຈະກໍາ** ຢູ່ດ້ານຂວາຂອງລາຍການ

## 5. ຕົວຢ່າງ & ການສະແດງຜົນ UI

### 5.1 ຕົວຢ່າງ 1: ສ້າງລາຍໄດ້ປົກກະຕິລາຍເດືອນ (ເງິນເດືອນ)

**ສະຖານະການ**: ທ່ານຕ້ອງການຕິດຕາມເງິນເດືອນລາຍເດືອນເພື່ອໃຫ້ແອັບອັດຕະໂນມັດແຈ້ງເຕືອນທ່ານເມື່ອເຖິງເວລາຮັບເງິນ.

**ຂັ້ນຕອນ**:
1. ໄປທີ່ຫນ້າຈໍຟັງຊັນ, ເລືອກ "ລາຍໄດ້ປົກກະຕິ"
2. ແຕະປຸ່ມ "➕ ເພີ່ມໃໝ່" ຢູ່ມຸມລຸ່ມຂວາ
3. ເລືອກປະເພດ "ເງິນເດືອນ" (ຫຼືສ້າງໃໝ່ຖ້າບໍ່ມີ)
4. ໃສ່ຈໍານວນ: 80,000,000 ກີບ
5. ເລືອກວົງຈອນ "ລາຍເດືອນ"
6. ເລືອກ "ເລືອກວັນຂອງເດືອນ", ໃສ່ 5
7. ໝາຍເຫດອັດຕະໂນມັດຕື່ມ "ເງິນເດືອນລາຍເດືອນ" (ສາມາດແກ້ໄຂໄດ້)
8. ແຕະ "ບັນທຶກ"

**ຜົນລັບ**: ແອັບສະແດງຂໍ້ຄວາມສໍາເລັດແລະກັບໄປຫາລາຍການ. ລາຍການໃໝ່ປາກົດດ້ວຍຂໍ້ມູນເຕັມ, ແລະແອັບຈະອັດຕະໂນມັດແຈ້ງເຕືອນທ່ານໃນວັນທີ 5 ຂອງແຕ່ລະເດືອນ.

### 5.2 ຕົວຢ່າງ 2: ຢືນຢັນລາຍໄດ້ທີ່ຄວນໄດ້ຮັບແລະອັບເດດຈໍານວນຕົວຈິງ

**ສະຖານະການ**: ມື້ຊໍາລະເງິນ (ວັນທີ 5), ແຕ່ຈໍານວນຕົວຈິງທີ່ໄດ້ຮັບແມ່ນ 84,000,000 ກີບ (ເງິນເດືອນເພີ່ມ) ແທນທີ່ຈະເປັນ 80,000,000 ກີບທີ່ຕັ້ງໄວ້.

**ຂັ້ນຕອນ**:
1. ເປີດແອັບຫຼືໄປທີ່ຫນ້າຈໍ "ລາຍໄດ້ປົກກະຕິ"
2. ແອັບອັດຕະໂນມັດກວດພົບການເກີດຂຶ້ນທີ່ຄວນໄດ້ຮັບແລະສະແດງການສົນທະນາຢືນຢັນ
3. ການສົນທະນາສະແດງຈໍານວນເລີ່ມຕົ້ນ: 80,000,000 ກີບ
4. ອັບເດດຈໍານວນຕົວຈິງເປັນ 84,000,000 ກີບ
5. ໃສ່ໝາຍເຫດ: "ເດືອນນີ້ມີໂບນັດ" (ທາງເລືອກ)
6. ແຕະ "ຢືນຢັນໄດ້ຮັບ"

**ຜົນລັບ**: ແອັບອັບເດດການເກີດຂຶ້ນທີ່ຢືນຢັນດ້ວຍຈໍານວນຕົວຈິງ 84,000,000 ກີບ, ອັດຕະໂນມັດສ້າງການເກີດຂຶ້ນຕໍ່ໄປ, ແລະອັບເດດຍອດເງິນປັດຈຸບັນ.

**ການສົນທະນາຢືນຢັນລາຍໄດ້**:

### 5.3 ຕົວຢ່າງ 3: ຍົກເລີກການເກີດຂຶ້ນລາຍໄດ້ເມື່ອບໍ່ໄດ້ຮັບເງິນ

**ສະຖານະການ**: ມື້ຊໍາລະເງິນເຊົ່າ (ວັນທີ 1), ແຕ່ຜູ້ເຊົ່າຍັງບໍ່ໄດ້ໂອນເງິນດັ່ງນັ້ນບໍ່ໄດ້ຮັບເງິນ.

**ຂັ້ນຕອນ**:
1. ເປີດແອັບຫຼືໄປທີ່ຫນ້າຈໍ "ລາຍໄດ້ປົກກະຕິ"
2. ແອັບສະແດງການສົນທະນາຢືນຢັນສໍາລັບການເກີດຂຶ້ນທີ່ຄວນໄດ້ຮັບ
3. ແຕະປຸ່ມ "ຍົກເລີກນີ້"
4. ໃສ່ເຫດຜົນ: "ຜູ້ເຊົ່າຍັງບໍ່ໄດ້ໂອນເງິນ" (ຈໍາເປັນ)
5. ແຕະ "ຢືນຢັນຍົກເລີກ"

**ຜົນລັບ**: ການເກີດຂຶ້ນທີ່ຍົກເລີກປ່ຽນເປັນສະຖານະ "ຍົກເລີກ", ສະແດງເຫດຜົນການຍົກເລີກ, ແລະແອັບອັດຕະໂນມັດສ້າງການເກີດຂຶ້ນຕໍ່ໄປ. ຍອດເງິນບໍ່ປ່ຽນແປງເພາະບໍ່ໄດ້ຮັບເງິນ.

**ການສົນທະນາຍົກເລີກການເກີດຂຶ້ນລາຍໄດ້**:

```text
┌─────────────────────────────────────────┐
│  ຍົກເລີກນີ້                            │
├─────────────────────────────────────────┤
```

## 6. ເຫດຜົນ & ກົດລະບຽບ

### 6.1 ວົງຈອນແລະວັນທີ

- **ອາທິດ**: ເລືອກວັນຂອງອາທິດ (1=ຈັນ, 7=ອາທິດ)
- **ສອງອາທິດ**: ເລືອກວັນຂອງອາທິດ + ວັນທີເລີ່ມຕົ້ນສະເພາະ
- **ລາຍເດືອນ**: ເລືອກວັນຂອງເດືອນ (1-31)

### 6.2 ສ້າງການເກີດຂຶ້ນອັດຕະໂນມັດ

- ແອັບສ້າງ **ການເກີດຂຶ້ນ** ອັດຕະໂນມັດເມື່ອ:
  - ເພີ່ມລາຍໄດ້ໃໝ່
  - ເຖິງວັນທີໃນວົງຈອນ
  - ເດືອນໃໝ່ເລີ່ມຕົ້ນ

### 6.3 ສະຖານະການເກີດຂຶ້ນ

- **PENDING**: ລໍຖ້າການຢືນຢັນ (ສະແດງ badge ສີເຫຼືອງ)
- **COMPLETED**: ຢືນຢັນແລ້ວ (ສະແດງ badge ສີຂຽວ)
- **CANCELLED**: ຍົກເລີກ (ສະແດງ badge ສີແດງ)

### 6.4 ການປະສົມປະສານກັບງົບປະມານ

- ເມື່ອຢືນຢັນລາຍໄດ້, ແອັບອັດຕະໂນມັດອັບເດດງົບປະມານເດືອນປັດຈຸບັນ (ຖ້າມີ)
- ລາຍໄດ້ຖືກຄິດໄລ່ເຂົ້າໃນ "ລາຍໄດ້ປົກກະຕິ" ໃນງົບປະມານ

### 6.5 ການແຈ້ງເຕືອນ

- ແອັບສົ່ງການແຈ້ງເຕືອນເມື່ອເງິນຄວນໄດ້ຮັບ
- ເວລາແຈ້ງເຕືອນສາມາດຕັ້ງຄ່າໄດ້ສໍາລັບແຕ່ລະລາຍການ (`notificationTime1`, `notificationTime2`, ເລີ່ມຕົ້ນ 16:00 ແລະ 19:00)

## 7. ໝາຍເຫດສໍາຄັນ

- **ຈໍານວນສາມາດປ່ອຍໄວ້ວ່າງ**: ຖ້າທ່ານບໍ່ຮູ້ຈໍານວນທີ່ແນ່ນອນ, ທ່ານສາມາດປ່ອຍໄວ້ວ່າງແລະໃສ່ເມື່ອຢືນຢັນ
- **ບໍ່ສາມາດລຶບເມື່ອມີການເກີດຂຶ້ນ**: ຖ້າມີການເກີດຂຶ້ນ, ທ່ານສາມາດປິດໃຊ້ງານເທົ່ານັ້ນ (isActive = false), ບໍ່ສາມາດລຶບ
- **ການຢືນຢັນຊ້າ**: ທ່ານສາມາດຢືນຢັນການເກີດຂຶ້ນໃນອະດີດ, ແອັບຈະອັດຕະໂນມັດຄິດໄລ່ງົບປະມານໃໝ່
- **ປ່ຽນວົງຈອນ**: ເມື່ອແກ້ໄຂວົງຈອນ, ການເກີດຂຶ້ນໃນອະນາຄົດຈະຖືກຄິດໄລ່ໃໝ່
