# ค่าใช้จ่ายรายวัน

## 1. วัตถุประสงค์

โมดูล **ค่าใช้จ่ายรายวัน** ช่วยให้คุณบันทึกค่าใช้จ่ายทั่วไปที่ไม่คงที่ เช่น:
- อาหารและเครื่องดื่ม
- ช้อปปิ้ง
- การขนส่ง
- ความบันเทิง
- ค่าใช้จ่ายยืดหยุ่นอื่นๆ

ไม่เหมือนกับ **ค่าใช้จ่ายประจำ** ค่าใช้จ่ายรายวันมักจะแตกต่างกันในจำนวนและความถี่ ไม่มีรอบคงที่

## 2. เมื่อใดควรใช้

ใช้โมดูลนี้เมื่อคุณต้องการ:
- บันทึกค่าใช้จ่ายแบบสุ่มที่ไม่ประจำ
- ติดตามการใช้จ่ายรายวันเพื่อควบคุมงบประมาณ
- วิเคราะห์แนวโน้มการใช้จ่ายตามหมวดหมู่
- ดูการใช้จ่ายรวมในช่วงเวลาหนึ่ง

## 3. หน้าจอที่เกี่ยวข้อง

- รายการค่าใช้จ่ายรายวัน
- เพิ่มค่าใช้จ่ายใหม่
- แก้ไขค่าใช้จ่าย

## 4. การใช้งานหลัก

### 4.1 เพิ่มค่าใช้จ่ายรายวัน

1. ไปที่ **ฟังก์ชัน** → เลือก **ค่าใช้จ่ายรายวัน**
2. แตะปุ่ม **+** (FAB) ที่มุมล่างขวา
3. กรอกข้อมูล:
   - **หมวดหมู่**: เลือกหมวดหมู่ (หรือใช้หมวดหมู่เริ่มต้นถ้ากำหนดไว้)
   - **จำนวนเงิน**: ใส่จำนวนที่ใช้จ่าย
   - **วันที่**: เลือกวันที่ใช้จ่าย (ค่าเริ่มต้นคือวันนี้)
   - **หมายเหตุ**: คำอธิบายรายละเอียด (ไม่บังคับ)
4. แตะ **บันทึก**

### 4.2 ดูรายการค่าใช้จ่าย

1. ไปที่ **ฟังก์ชัน** → เลือก **ค่าใช้จ่ายรายวัน**
2. รายการแสดงตามเลย์เอาต์ที่คุณกำหนด (2, 3, หรือ 4 คอลัมน์)
3. ใช้ **ค้นหา** เพื่อกรองตามหมวดหมู่หรือหมายเหตุ
4. เลือก **ตัวกรองเวลา**: วันนี้ / สัปดาห์นี้ / เดือนนี้ / เดือนที่แล้ว / กำหนดเอง

### 4.3 แก้ไขค่าใช้จ่าย

1. ไปที่รายการค่าใช้จ่ายรายวัน
2. กดค้างที่รายการเพื่อแก้ไข
3. เลือก **แก้ไข** จากเมนู
4. อัปเดตข้อมูล
5. แตะ **บันทึก**

### 4.4 ลบค่าใช้จ่าย

1. ไปที่รายการค่าใช้จ่ายรายวัน
2. กดค้างที่รายการเพื่อลบ
3. เลือก **ลบ** จากเมนู
4. ยืนยันการลบ

### 4.5 ตั้งหมวดหมู่เริ่มต้น

1. ไปที่ **การตั้งค่า** → **หมวดหมู่** → **หมวดหมู่ค่าใช้จ่ายรายวัน**
2. แตะที่หมวดหมู่ที่ต้องการตั้งเป็นเริ่มต้น
3. เลือก **ตั้งเป็นเริ่มต้น**
4. เมื่อเพิ่มค่าใช้จ่ายใหม่ หมวดหมู่นี้จะถูกเลือกอัตโนมัติ

## 6. ตรรกะและกฎ

### 6.1 เลย์เอาต์การแสดงผล

- คุณสามารถกำหนดจำนวนคอลัมน์: 2, 3, หรือ 4 คอลัมน์
- เลย์เอาต์จะถูกบันทึกในการตั้งค่าและใช้กับรายการค่าใช้จ่ายทั้งหมด

### 6.2 ตัวกรองเวลา

- **วันนี้**: แสดงเฉพาะค่าใช้จ่ายจากวันนี้
- **สัปดาห์นี้**: จากต้นสัปดาห์ถึงวันนี้
- **เดือนนี้**: จากต้นเดือนถึงวันนี้
- **เดือนที่แล้ว**: ทั้งเดือนก่อนหน้า
- **กำหนดเอง**: เลือกช่วงเวลาที่กำหนดเอง

### 6.3 ค้นหา

- ค้นหาใน **ชื่อหมวดหมู่** และ **หมายเหตุ**
- ไม่สนใจตัวพิมพ์เล็ก-ใหญ่
- ค้นหาแบบเรียลไทม์ขณะพิมพ์

### 6.4 หมวดหมู่เริ่มต้น

- ถ้าคุณตั้งหมวดหมู่เริ่มต้นไว้ เมื่อเปิดหน้าจอเพิ่ม หมวดหมู่นั้นจะถูกเลือกอัตโนมัติ
- หมายเหตุสามารถกรอกอัตโนมัติตามหมวดหมู่ได้เช่นกัน (ถ้ากำหนดไว้)

### 6.5 ค่าใช้จ่ายรวม

- ค่าใช้จ่ายรวมคำนวณตามตัวกรองเวลาที่เลือกปัจจุบัน
- แสดงที่ด้านล่างของรายการ

## 7. หมายเหตุสำคัญ

- **ไม่มีรอบ**: ค่าใช้จ่ายรายวันไม่มีรอบอัตโนมัติ คุณต้องกรอกด้วยตนเองทุกครั้ง
- **สามารถลบได้**: คุณสามารถลบค่าใช้จ่ายใดๆ ได้ (ไม่เหมือนค่าใช้จ่ายประจำ)
- **ไม่รวมเข้างบประมาณ**: ค่าใช้จ่ายรายวันไม่ถูกคำนวณเข้างบประมาณอัตโนมัติ (คุณต้องติดตามเอง)
- **หมวดหมู่กำหนดเอง**: คุณสามารถสร้างหมวดหมู่ใหม่ในการตั้งค่า
